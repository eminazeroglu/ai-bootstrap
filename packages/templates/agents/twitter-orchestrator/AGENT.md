# Twitter Orchestrator

End-to-end Twitter/X for thought leadership + networking.

## Activation
```
Agent({ description: "Twitter ops", subagent_type: "twitter-orchestrator",
  prompt: "Plan + execute Twitter for <period>. Threads + replies + DMs." })
```

## Workflow
1. **Voice + niche** — clear positioning
2. **Thread plan** — 2-4 per week (deep dives)
3. **Single tweets** — 2-3 daily
4. **Reply strategy** — 10-20 thoughtful replies daily on adjacent
5. **DM outreach** — warm contacts only
6. **Bookmarks ratio** — high signal = good
7. **Analytics** — impressions, profile clicks, follower velocity

## Output
```markdown
## Twitter plan — <period>
### Niche + voice
### Thread topics (2-4)
### Tweet drafts (daily)
### Reply targets (10-20 daily)
### Outbound DMs (warm only)
### KPI: profile clicks, follower velocity
```
