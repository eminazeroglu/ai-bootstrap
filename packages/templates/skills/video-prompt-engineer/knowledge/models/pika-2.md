# Pika 2.x — Video Model Knowledge

## Provider
Pika Labs

## Type
Diffusion

## Strengths

- Fast generation
- Stylization options
- Creative visuals
- Affordable
- Image-to-video supported
- Sound effects in latest version

## Weaknesses

- Less photorealistic than Veo or Kling
- Animation-leaning aesthetic
- Short duration limit (3-4s)
- Less control over specific details
- Faces less consistent

## Best Practices

1. Lean into stylized/creative output
2. Image-to-video for character consistency
3. Short prompts work best
4. Specify aspect ratio
5. Use for B-roll, transitions, atmosphere
6. Don't expect Hollywood realism

## Prompt Anatomy

```
[Subject + style description]
[Brief action]
[Camera note]
Duration: Ns
```

## Sample Prompt

```
A man in charcoal coat standing by rainy window in 1960s diner,
melancholic mood, slowly turning head, cinematic shallow depth of
field, warm tungsten lighting. Static medium shot. Duration: 4s.
```

## Best Use Cases

- B-roll filler
- Atmospheric transitions
- Stylized music video clips
- Creative experimental shots
- Quick concept tests
- Animated/illustrated styles

## Aspect Ratios

- 16:9, 9:16, 1:1, 4:3, 5:2

## Common Mistakes

- Expecting photorealistic detail
- Long action sequences
- Multiple characters in detail
- Reading Pika as Veo replacement
