# Cinema Theatrical — Platform

> Big screen theatrical release. The traditional cinema experience.

## Defining characteristics

- Big screen (theater)
- No interruptions / breaks
- Captive audience (paid for ticket)
- Premium production value expected
- Cinematic aspect ratios (1.85:1, 2.39:1)
- Surround sound design
- Long-form (90-180 min typical)

## Format conventions

### Length
- Short film festival: 1-30 min
- Indie feature: 75-100 min
- Mainstream feature: 100-150 min
- Epic / prestige: 150-210 min

### Aspect ratios
- 1.85:1 — standard widescreen
- 2.39:1 — anamorphic / cinemascope
- 1.66:1 — European art-house
- 1.43:1 — IMAX (rare)
- Aspect impacts framing fundamentally

### Pacing
- Patient (audience can't skip)
- Long takes welcome
- Silence has weight
- Build to crescendo

## Writing for cinema

### Visual storytelling first
- Trust the image
- Less dialogue, more action/expression
- Cinematic shots designed for big screen
- Spectacle moments (vista, scale)

### Sound design
- Surround dimensionality
- Silence vs. sound contrast
- Music score as character

### Theatrical experience
- Audiences experience together
- Communal laughter, gasps
- Build for these moments
- Don't talk DOWN to audience

### Major studio vs indie

**Major studio (Hollywood)**
- 90-130 min commercial
- Conventional structure
- Genre adherence
- Star vehicles
- Marketable hooks

**Indie / Festival**
- Length flexible
- Experimental structure ok
- Auteur voice
- Character-driven
- Theme over plot

**Art house / International**
- Slow cinema acceptable
- Long takes
- Foreign language
- Subversive narrative
- Limited release expected

## Genre considerations

### Action / Spectacle
- Set pieces every 15-20 min
- Big screen-worthy visuals
- Practical effects ideal

### Drama
- Character peaks at 30, 60, 90 min
- Emotional crescendos
- Long performance moments

### Comedy
- Audience reaction breaks
- Allow laughter to land
- Bigger gestures

### Horror
- Theater dark = jumps land harder
- Sound design crucial
- Patience for tension

### Animation
- Visual feast
- Character clarity (back row reads)
- All ages awareness

## Sample structure: 110-min feature

```
ACT 1 (0-25 min)
- Setup world, character
- Inciting incident around 10-15 min
- First-act break around 25 min

ACT 2A (25-55 min)
- Confrontation
- New world/journey
- Subplots introduced

MIDPOINT (55 min)
- Major reversal
- Rules change
- Stakes raised

ACT 2B (55-85 min)
- Continued struggle
- Crisis builds
- All-is-lost moment around 80 min

ACT 3 (85-110 min)
- Final attempt
- Climax around 95-100 min
- Resolution / new normal
```

## Pitfalls

- TV pacing (too fast for big screen)
- Streaming-optimized hooks (cinema audience already invested)
- Dialogue-heavy when visual would work
- Forgetting cinema sound (silent moments waste)
- Aspect ratio mismatch

## When to write for cinema

- Theatrical release planned
- Festival submission
- Premium production
- Long-form story requiring big screen
- Visual scale that needs theater

## Modern context (2026)

Cinema-only content rare; most has streaming destination too. But cinema-FIRST writing:
- Spectacle
- Communal experience moments
- Long-form character investment
- Not optimized for second-screen viewing
