# Character Arcs — Script Core

> How characters change over the course of a story. Even short formats need arcs.

## What is a character arc

A character arc = the internal journey from one state to another:
- Belief → New belief
- Flaw → Healing (or destruction)
- Want → Need (recognized or rejected)

## Arc types

### Positive arc (growth)
Character starts flawed, ends healed.
- Example: Selfish protagonist learns sacrifice
- Most common in mainstream

### Negative arc (fall)
Character starts okay, ends worse.
- Example: Walter White becomes Heisenberg
- Tragedy structure

### Flat arc (steadfast)
Character stays the same; world changes around them.
- Example: Indiana Jones, Atticus Finch
- They influence others to change

### Disillusionment arc
Character believes lie, learns truth.
- Coming-of-age stories

## Arc components

### 1. Initial state
- Where character starts
- Their flaw / wound
- Their belief about world

### 2. Inciting incident
- Event that disrupts initial state
- Forces character into action

### 3. False victories / setbacks
- Character tries old ways → fails
- Or tries new ways → succeeds temporarily

### 4. Crisis / dark night
- Lowest point
- Old way fully fails
- Character must change or be destroyed

### 5. Choice
- Character chooses new way (positive)
- Or chooses old way (negative/tragic)

### 6. Resolution
- New state established
- Echo of beginning showing change

## Arc in different lengths

### TikTok storytime (60s)
Even here, an arc:
- 0:00-0:10: Setup ("I thought I'd be alone forever")
- 0:10-0:30: Escalation (event happens)
- 0:30-0:50: Realization (insight)
- 0:50-1:00: Change (new state)

### 30s ad
Often character/customer arc:
- Has problem → discovers product → transformed

### Short film
Single clear arc, focused
- One want/need
- One major change

### Series episode
Episode-level arc + season arc
- Each episode has self-contained arc
- Season builds larger transformation

### Feature film
Major arc + subplot arcs
- Protagonist's main arc
- Supporting character arcs (smaller)

## The "3 dimensions" of character

### 1. Want (external goal)
What they're chasing on screen.

### 2. Need (internal truth)
What would actually fulfill them.

### 3. Wound (past hurt)
Why they have the flaw they have.

Best stories: All three connect.
- Want = "Get the promotion"
- Need = "Self-worth not from external validation"
- Wound = "Father never said he was proud"

## Setup → payoff for arcs

### Setup arc clearly
Show character's flaw EARLY through action:
```
EARLY SCENE: ƏLİ refuses help carrying groceries.
"I've got it."
He drops everything trying to prove he doesn't need help.
```

### Pay off arc clearly
Show flaw resolved through opposite action:
```
LATE SCENE: ƏLİ asks for help with the moving van.
"I can't do this alone."
LEYLA helps. They smile.
```

This proves change without saying it.

## Avoid

### Static characters
Character ends exactly where they started. Boring.

### Unmotivated change
Character changes because plot requires it, not because earned.

### On-the-nose declarations
```
BAD:
ƏLİ
I've learned that I need to ask for help!
```

Show through action, not statement.

### Multiple arcs without focus
One main arc, supporting beats. Not three competing arcs.

## Arc + plot relationship

- Plot = external events
- Arc = internal change
- Best stories: external plot FORCES internal change

Plot without arc = action with no meaning.
Arc without plot = navel-gazing.

## Quick check

For any story, answer:
1. Where does protagonist start (state, belief, flaw)?
2. What event forces change?
3. What do they try that doesn't work?
4. What's the crisis?
5. Where do they end up?
6. How is the ending DIFFERENT from beginning?

If you can't answer all, the arc is unclear.
