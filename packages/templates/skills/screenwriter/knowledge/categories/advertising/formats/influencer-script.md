# Influencer Script — Format

> Brand sponsorship integrated into influencer's native content. Their voice, their audience.

## Format basics

- Length: 30s - 90s (within longer content sometimes)
- Aspect: matches platform (9:16 mobile mostly)
- Influencer's native style
- Brand mentioned naturally
- Disclosure required (#ad)

## What this is

NOT a brand commercial. An influencer's regular content with brand integrated. Different per influencer:
- Beauty: tutorial featuring product
- Fitness: workout using product
- Tech: review or workflow
- Lifestyle: day-in-life with product
- Food: recipe featuring product

## Structure (varies by influencer, but typical)

```
0-5s    HOOK (their style)
        Their typical content opener

5-15s   CONTEXT
        Setting up the topic/situation

15-50s  CONTENT + BRAND INTEGRATION
        Their normal content
        Brand naturally appears
        Authentic use

50-58s  ENDORSEMENT MOMENT
        Why they personally like it
        Honest assessment
        Their audience-specific angle

58-60s  CTA
        Discount code
        Link in bio
        Brand tag
```

## Sample 60s influencer script

```

(55-60s)

ON SCREEN: #ad #sponsored #karavan link in bio
```

## Key principles

### Influencer voice intact
The brand integrates into the CREATOR's content style:
- Their typical opening
- Their typical pacing
- Their typical visual style
- Their typical CTA approach

DON'T impose brand voice over influencer voice.

### Authentic-feeling endorsement
- Honest reaction (not fake-enthusiastic)
- Specific to their content niche
- "Why I personally use this"
- Critical edge ok ("It's not perfect but...")

### Disclosure
- Disclosure must be visible/audible
- Platform-appropriate (#ad, "sponsored", etc.)
- Doesn't kill content if integrated well

### Discount code / tracking
Most influencer deals have:
- Unique code (BAKU10)
- Trackable link
- Bio link mention
- Affiliate structure

## Sub-types

### Beauty influencer
- Tutorial featuring product
- Result demonstration
- Comparison to other products

### Fitness influencer
- Workout using product (supplement, gear)
- Performance angle
- Recovery/lifestyle integration

### Tech influencer
- Review or workflow integration
- Honest pros/cons
- Setup/unboxing

### Food influencer
- Recipe featuring product
- Behind-scenes prep
- Meal context

### Lifestyle influencer
- Day-in-life
- Routine integration
- Aspirational context

## Best practices

### Brief, not script
Most influencer deals: brief + bullet points, NOT verbatim script. Influencer adapts to their voice.

### Authenticity > polish
Audience knows when something is off. Authentic feels mandatory.

### Audience match
Brand should genuinely fit the influencer's audience. Otherwise rejection.

### Soft sell
- "I'm using this"
- "Worth trying"
- NOT "You must buy this!"

## Common mistakes

- Brand language imposed on creator
- Over-promising
- Awkward integration (forced product placement)
- Skipping disclosure
- Wrong audience match
- Robot-script reading

## Compliance

- FTC disclosure (US)
- Local regulations (each country)
- Platform-specific requirements
- Always #ad / "sponsored" / clear identifier

## When to use

- Building trust through known voice
- Reaching specific audience niche
- Native content beats banner ads
- Long-term brand partnerships
- Product launches via influencer reach
