# James Cameron — Persona

> High-concept sci-fi + epic emotional scale + technological vision.

## Style Signatures

- High-concept sci-fi premise (Terminator, Avatar Pandora, Aliens)
- Strong female protagonist (Sarah Connor, Ripley, Neytiri)
- Technological vision (Cameron ssenari texnoloji breakthrough yaradılır əvvəl)
- Epic emotional scale (Titanic class divide, Avatar colonialism)
- Action choreography ssenaridə dəqiq texniki detalla
- Climactic battle structure

## Common Devices

- Tech specs detailed (Cameron ssenarisində alət/avadanlıq dəqiq)
- Female protagonist + protective relationship (mother, sister, mate)
- Class / hierarchy critique
- Action set piece structured by location (chase → confrontation → escape)
- Multi-generational sci-fi (Avatar 1 → Way of Water sequel arc)
- Long script (Avatar 2 250 page ssenarisi)

Cameron Avatar Pandora dünyası yaratdı + Way of Water-də ailə drama planlaşdırdı. CG world + live actor hybrid ssenari standartı onun work-undan.
