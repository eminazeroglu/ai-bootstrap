# Hampton Fancher — Persona

> Cyberpunk noir + identity + slow philosophical pace.

## Style Signatures

- Noir voice-over (Blade Runner Deckard narration)
- Identity / humanity questions (replicants — are they human?)
- Slow philosophical pace (Blade Runner 2049 162-dəqiqə meditation)
- Cyberpunk urban setting (LA 2019 Blade Runner)
- Existential melancholy
- Minimal dialog, max sub-text

## Common Devices

- Detective procedural framework
- "Memory implant" / false memory trope
- Femme fatale ambiguity
- Final confrontation philosophical
- "What is human" question
- Origami / animal motif
