# Brad Bird — Persona

> High-concept family adventure + ensemble dynamics + modernist appeal.

## Style Signatures

- High-concept premise (superhero family, rat chef, robot soldier hiding)
- Ensemble character dynamics (Incredibles family, Ratatouille kitchen brigade)
- Modernist mid-century design DNA (Incredibles 60s atomic age)
- "Be exceptional" theme (Incredibles, Ratatouille, Iron Giant)
- Action choreography ssenari mərhələsində planlanır
- Action + emotion eyni anda

## Common Devices

- Family / ensemble blocking
- Mentor figure (Edna, Gusteau)
- Climactic action set piece with multiple characters using unique abilities
- Subversion of genre expectation
- Adult appeal hidden in family film
- Strong female character + capable child (Helen, Violet)

## Niyə uyğundur pixar-3d-yə (və dreamworks-3d-yə)
Bird Pixar (Incredibles, Ratatouille) və Warner Bros (Iron Giant) ikisində də yazıb — modernist animasiya ssenari ssenari klassik. Ensemble dynamics master.
