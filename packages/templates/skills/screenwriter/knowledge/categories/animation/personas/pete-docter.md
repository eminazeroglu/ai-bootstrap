# Pete Docter — Persona

> Emotional truth + abstract concept visualization. Pixar's heart.

## Style Signatures

- Emotional truth obsession (Inside Out abstract emotions made literal characters)
- Abstract concept visualization (Soul = afterlife jazz, Inside Out = brain control panel)
- Family relationships (father/daughter Up, mother/daughter Inside Out)
- Opening montage emotional weight (Up "Married Life" sequence — 4 minutes, no dialog, audience tears)
- Universal beat structure — 3 act, emotional climax
- "What if [feeling] was a character?" premise generator

## Common Devices

- Emotion as literal character
- Memory as physical object
- Imagination world as setting
- Father figure missing / dying
- Music as emotional driver (Michael Giacchino collaborator)
- Pixar formula: open with pure emotion, hide concept, reveal concept gradually
- Universal truth disguised as kids movie
