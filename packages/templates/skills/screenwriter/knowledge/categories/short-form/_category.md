# Short-Form — Category

> Vertical short-form social media content. TikTok, Reel, YouTube Shorts. Maximum hook, minimum runtime.

## What short-form is

Mobile-first vertical video, typically 15-90 seconds. Algorithm-driven discovery. Sound-on default. Native to platform conventions.

## Universal short-form principles

### 1. Hook in 1-3 seconds
If audience swipes past in first 3 seconds, video is dead. Hook must be immediate.

### 2. Vertical format (9:16)
Mobile-first. Don't crop landscape — film vertical.

### 3. Sound-on design
Music + voice + sound effects layered. Captions for accessibility.

### 4. Native to platform
TikTok ≠ Reel ≠ YouTube Shorts. Each has subtle conventions.

### 5. High pacing
Cut every 1-3 seconds. No dead air. Information density maximum.

## Universal structure

```
0-3s    HOOK
        Visual + audio grab
        Question / claim / image

3-15s   SETUP
        Context / problem
        Tension build

15-50s  PAYOFF / CONTENT
        The promise delivered
        Story or value

50-60s  CTA / RESOLUTION
        Follow / link / reaction prompt
        Memorable close
```

## Length sweet spots

| Length | Use case |
|--------|----------|
| 7-15s | Quick joke, single moment |
| 15-30s | Trend, simple narrative |
| 30-60s | Storytime, tutorial, sketch |
| 60-90s | Deep storytime, educational |
| 90-180s | Multi-part stories |
| 3-10min | Educational deep dive (rare) |

## Voice styles

### Native creator voice
Direct address, casual, vulnerable, native vocabulary

### Storyteller
Personal narrative, hook-and-resolve

### Educational
Clear teaching, structured, trust-building

### Entertainment
High energy, joke-dense

### Aesthetic
Mood-driven, music-first, low-dialogue

## Common mistakes

- Slow opening (3s rule)
- Generic hooks
- Trend-late
- Too polished (lacks authenticity)
- Long talking-head openings
- Forgetting captions
- Wrong platform conventions
