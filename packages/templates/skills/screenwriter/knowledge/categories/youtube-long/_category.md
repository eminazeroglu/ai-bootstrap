# YouTube Long-Form — Category

> Long-form video content on YouTube. 5-60+ minutes. Algorithm + retention driven.

## What YouTube long is

Standard YouTube videos: 5-60+ minutes. Distinguished from Shorts (<60s). Distinguished from streaming TV (no episode/season structure). Direct creator-audience relationship.

## Universal long-form principles

### 1. Hook in first 15-30 seconds
Retention is everything. Hook hard or audience leaves.

### 2. Pattern interrupts every 60-90s
Visual variety, cuts, location changes — break monotony.

### 3. Promise + delivery
Title + thumbnail = promise. Video must deliver.

### 4. Personality-driven
YouTube creators are PERSONALITIES. Brand built on you.

### 5. CTA at end
Subscribe, like, comment, next video.

## Length conventions

| Type | Length |
|------|--------|
| Quick tutorial | 3-7 min |
| Standard video | 8-15 min |
| Deep dive / essay | 15-30 min |
| Documentary / commentary | 30-60+ min |
| Podcast / interview | 60-120+ min |

## Algorithm awareness

YouTube rewards:
- Watch time (total minutes watched)
- Average view duration (% completion)
- Click-through rate (CTR) on thumbnail
- Engagement (likes, comments, shares)
- Session time (do viewers watch more YT after)

Each script choice should serve these metrics.

## Common pitfalls

- Slow opening
- No pattern interrupts (boring middle)
- Forgetting subscribe CTA
- Weak title/thumbnail (low CTR)
- Inconsistent voice
- Misleading title (audience leaves angry)
