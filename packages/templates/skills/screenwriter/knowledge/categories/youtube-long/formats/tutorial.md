# Tutorial — YouTube Long Format

> "How to ___" video. Educational. Step-by-step. Result-focused.

## Format basics

- Length: 8-20 min typical
- Aspect: 16:9
- Step-by-step structure
- Visual demonstrations
- Result shown clearly
- Educational value primary

## Structure

```
0-30s    HOOK
         What you'll learn
         Result preview

30s-2min INTRO
         Why this matters
         Who it's for

2-15min  STEPS
         Numbered or sequential
         Each step demonstrated
         Common mistakes addressed

15-18min RESULT / DEMONSTRATION
         Final outcome shown
         Quality demonstrated

18-20min CTA + RECAP
         Subscribe
         Quick recap of key points
```

## Sample 12-min tutorial

```
"How to Make Café-Quality Pour-Over Coffee at Home"

0-30s: HOOK
- Beautiful pour-over coffee being finished
- "Today I'll teach you to make pour-over coffee that beats your
local café."

30s-2min: INTRO
- Why pour-over is best for taste
- Equipment needed (shown clearly)
- Time investment (15 minutes)

2-3min: STEP 1 — EQUIPMENT
- Show all gear: dripper, filter, kettle, scale, grinder, mug
- Total cost breakdown
- Where to buy each (no specific brand promotion)

3-5min: STEP 2 — BEAN SELECTION
- Fresh, whole, single-origin
- How to read roast date
- Storage tips

5-7min: STEP 3 — GRIND
- Why grind matters
- Medium-coarse like sea salt
- Visual demonstration

7-9min: STEP 4 — BLOOM
- Pre-wet filter
- Add grounds
- 50ml water, 30 second wait
- Why this matters (CO2 release)

9-11min: STEP 5 — POUR
- Slow spiral from center outward
- Maintain water level
- Total pour time: 3 minutes

11-12min: TASTING + RESULT
- Pour into cup
- Sip, savor
- Tasting notes (this blend: stone fruit + dark chocolate)

12-12:30: CTA
- "Save this video for your morning routine"
- "Subscribe for more home barista tips"
- "Next week: how to choose the right beans"
```

## Key principles

### Result-first
- Show what they'll achieve early
- Build desire to learn

### Numbered structure
- Easy to follow
- Easy to bookmark
- Save-friendly

### Visual demonstration
- Don't just talk
- Show every step
- Multiple angles

### Address common mistakes
- "Don't do this..."
- Pre-empt questions

### Time-realistic
- Don't make it look easier than it is
- Acknowledge difficulty

## Sub-types

### Software tutorial
- Screen recording dominant
- UI walkthrough
- Specific outcomes

### Craft tutorial
- Hands-on demonstration
- Tools shown
- Skill development

### Recipe tutorial
- Step-by-step cooking
- Ingredients clear
- Final dish reveal

### DIY tutorial
- Building something
- Tools + materials
- Result demonstration

### Skill tutorial
- New technique
- Practice progression
- Mastery promise

## When to use

- Authority building
- SEO-friendly content (search-driven traffic)
- Long-form companion to short content
- Brand education
- Audience trust building

## Common mistakes

- Skipping steps audience needs
- Too rushed (audience must follow)
- Missing result demonstration
- No common mistakes addressed
- Generic content (could be from anyone)

## Production tips

- Multiple camera angles
- Top-down for hands-on
- Clear audio
- Text overlays for steps
- Resource list in description
