# Sketch (Long) — YouTube Long Format

> Long-form comedy sketch. 5-15 min typical. Multiple characters, multiple beats.

## Format basics

- Length: 5-15 min typical
- Aspect: 16:9
- Multiple characters
- Comedic premise sustained
- Production value matters
- Cinematic feel often

## Structure

```
0-1min    SETUP
          Premise established
          Characters introduced

1-10min   ESCALATION
          Beats build
          Things go wrong (more wrong)
          Characters react

10-13min  CLIMAX
          Peak absurdity
          Big payoff

13-15min  RESOLUTION + TAG
          Aftermath
          Final joke
          CTA
```

## Sample 10-min sketch

```
"The World's Most Annoying Customer (At My Coffee Shop)"

0-1min: SETUP
- A normal coffee shop
- Barista (creator) prepping for day
- A CUSTOMER walks in

1-3min: BEAT 1 — The Order
- Customer: "I'd like a small coffee."
- Barista: "Coffee, sure. What kind?"
- Customer: "Just coffee."
- Barista (specific menu options)
- Customer (rejects each one for absurd reason)
- Eventually: "Just give me whatever you'd drink."
- Barista pours pour-over.

3-5min: BEAT 2 — The Customization
- Customer: "Can I add..."
- Asks for: oat milk (don't have it), specific honey, "exactly half a sugar packet"
- Barista accommodating
- Each request more absurd
- "Can the cup be slightly smaller? My hands are small."

5-7min: BEAT 3 — The Photo
- Customer wants to take photo
- "Could you steam the milk for the photo?"
- "Could you redo the latte art? It's not symmetrical."
- "Could the lighting be different?"
- Production: customer changes lighting, barista helps

7-9min: BEAT 4 — The Payment
- Customer doesn't have cash
- Card declined three times
- "Do you do tabs?"
- "Can I pay you back in coffee beans?"
- Negotiating

9-10min: CLIMAX + TAG
- Finally pays
- "This was great! Same time tomorrow?"
- Walks out happy
- Barista (to camera, dead-eyed): "It's only 8:15 AM."
- Cut to: clock on wall reading 8:15
- End on barista's defeated face
- TAG: Next customer walks in. Same energy. Barista's smile broken.

End on cut to black.
End card: "Subscribe for more shop suffering."
```

## Key principles

### Sustained premise
- One core absurdity
- Built through beats
- Each beat raises stakes

### Specific characters
- Distinct personalities
- Memorable quirks
- Comic timing

### Production value
- Looks intentional
- Lighting, audio, camera matter
- Don't look amateur (unless that's the joke)

### Escalation logic
- Each beat slightly worse
- Or absurdly worse
- Build to climax

### Earned punchline
- Setup pays off
- Audience feels rewarded
- Memorable

## Sub-types

### Single situation extended
- One scenario, multiple beats
- "What if ___ happened for 10 minutes"

### Day-in-life sketch
- Following character through day
- Multiple absurd moments
- Cumulative effect

### Multi-character sketch
- Ensemble piece
- Different characters
- Dynamics drive comedy

### Concept sketch
- "What if everyone in the world ___"
- Absurd premise played straight

### Parody
- Format / show / movie parodied
- Inside-joke for fans
- Plus broader comedy

## When to use

- Comedy creator content
- Audience entertainment
- Brand humor (with care)
- Building creator personal brand
- High production value showcase

## Common mistakes

- Premise unclear
- Stretches too thin
- Bad pacing
- Forgettable punchline
- Production values too low (looks bad)
- Length not earned
