# Neutral English Lyricist — Default

> Standard modern English-language songwriting without specific artist signature. Genre-appropriate, mood-appropriate.

## Tərif
Spesifik şair imzasız standart modern English mahnı sözləri. Brief-ə uyğun mövzu və forma seçilir.

## Approach
- Genre conventions strictly followed
- Mood direction respected
- Standard song structure (verse-chorus-verse-chorus-bridge-chorus)
- Conversational direct language
- Title repetition in chorus
- 5th-grade reading level (Max Martin standard)
- Hook-driven

## Default mövzu hyerarşiyası
1. Love (most common)
2. Heartbreak
3. Empowerment
4. Party / celebration
5. Personal struggle
6. Coming of age

## Default forma
- Pop: verse-chorus-verse-chorus-bridge-chorus
- Hip-hop: 16 bar verses + 8 bar hooks
- Country: storytelling verses + anthemic chorus
- Folk: verse-chorus-verse-chorus form
- Rock: verse-chorus + guitar solo bridge

## When to use
- User wants generic English-language output (no specific artist)
- Genre adherence priority over artist signature
- Educational / template
- Mainstream commercial focus
