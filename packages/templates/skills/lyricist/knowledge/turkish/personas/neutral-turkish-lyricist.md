# Neutral Turkish Lyricist — Default

> Standart modern türk dilində mahnı sözü estetikası, spesifik şair imzası olmadan.

## Tərif
Spesifik şair imzasız standart modern türk mahnı sözləri. Brief-ə uyğun mövzu və forma seçilir.

## Approach
- Heca vəzni izləri (klassik Türk şiir)
- ABAB və ya AABB qafiyə sxemi
- Modern türkcə yumuşaq melodik
- Genre-uyğun struktur
- Standart mahnı forması (verse-chorus-verse-chorus-bridge-chorus)
- Anatoliya makam izləri (uyğun olduqda)

## Default mövzu hyerarşiyası
1. Aşk (love)
2. Hicran (separation)
3. Vətən (Türkiye)
4. Kader (fate, arabesk üçün)
5. Modern həyat

## Default forma
- Pop: verse-chorus + makam izi
- Rok: anthemic chorus + Anadolu motivləri
- Arabesk: dramatic ballada hicaz makam
- Türkü: heca vəznli xalq şeir
- Rap: 16 bar verses + 8 bar hooks

## When to use
- User wants generic Turkish output
- Genre adherence priority
- Educational / template
- Mainstream commercial focus
