# İstiarə — Klassik Şərq Poeziyasının Gizli Köçürmə Texnikası

> Metaforanın xüsusi forması. Bir sözün başqa məna ilə "müvəqqəti istifadəsi". Sufi simvolizminin əsasıdır.

## Tərif
İstiarə — ərəb mənşəli, "ödünc almaq" mənasını verən termin. Şeirdə **bir sözü öz mənası yerinə başqa məna üçün müvəqqəti istifadə etməkdir**. Metaforanın qardaşıdır, lakin daha kompleks və klassik divan ədəbiyyatının səciyyəvi texnikasıdır.

## İstiarə vs Metafora vs Təşbeh

| | Təşbeh | Metafora | İstiarə |
|---|---|---|---|
| Müqayisə | Açıq ("kimi") | Gizli | Daha gizli, kompleks |
| Element | 4 element (müşəbbeh + müşəbbeh-bih + vəc-i şəbəh + ədat) | 2 element (köçürmə) | 1 element (sadəcə "müşəbbeh-bih") |
| Misal | "Yarımın gözü dəniz kimi" | "Yarımın gözü dəniz" | "Dəniz" (yar nəzərdə tutulur) |
| Klassik istifadə | Tez-tez | Tez-tez | Çox tez-tez (divan dominant) |

## İstiarənin əsas mexanizmi
- **Yalnız müşəbbeh-bih qalır**, müşəbbeh atılır
- Oxucu/dinləyici **kontekstdən** anlamalıdır müşəbbeh nədir

### Misal
- Real söz: "Mənim dilbərimi gördüm yenə də"
- Təşbeh: "Mənim dilbərimi (yarımı) **bir gül kimi** gördüm yenə də"
- Metafora: "Mənim dilbərimi **gül** gördüm yenə də"
- İstiarə: "**Gül** gördüm yenə də" (kontekstdə oxucu anlamalıdır ki, "gül" = sevgili)

## İstiarə növləri

### 1. Müsəmmaha istiarə (insan üçün təbiət obrazı istifadəsi)
- "Sərv" (sərv ağacı) → boylu sevgili
- "Bülbül" → aşiq özü
- "Şəm" (şam) → sevgili (cazibə)
- "Pərvanə" (kəpənək) → aşiq (alova vurğun)

### 2. Muqəyyəd istiarə (mücərrəd kimi yazılır)
- "Atəş" → eşq odu
- "Dəniz" → həyat
- "Şərab" → ilahi mərifət (sufi)
- "Meyxana" → tərəfdarlıq, sufi mərkəzi

### 3. Təbii istiarə (real obyektin başqası adına gətirilməsi)
- "Sərv qaməti" → sevgili boyu
- "Dəniz gözlü" → sevgili gözləri
- "Ay üzlü" → sevgili sifəti

### 4. Asılı istiarə (mücərrəd → mücərrəd)
- "Ümid çiçəyi" → ümid mücərrəd, çiçək metaforik

## Klassik divan ədəbiyyatında istiarələr

### Sevgili obrazları (istiarə şəklində)
- **Şəm** (şam) — sevgili (yandırıcı gözəllik)
- **Sərv** — sevgili boyu (sərv ağacı qədər uca)
- **Gül** — sevgili (gül qədər gözəl)
- **Mah** (ay) — sevgili sifəti
- **Bənövşə** — sevgili ürəyi
- **Sünbül** (sünbül) — sevgili saçları

### Aşiq obrazları
- **Bülbül** — aşiq (gülə vurğun)
- **Pərvanə** — aşiq (alova vurğun)
- **Məcnun** — divanə aşiq

### Mücərrəd istiarələr
- **Şərab** — sufi mərifət, ilahi eşq
- **Meyxana** — tərəfdarlıq, sufi mərkəzi
- **Saqi** — ilahi rəhbər, sevgili
- **Fənar** — qəlbin nuru, mərifət

## Misal: Füzuli istiarələri
```
Ey **gül**, niyə bağda yaln qaldın?     ← "gül" = sevgili
**Bülbül** gəldi yanına, sən gözünü çevirdin... ← "bülbül" = Füzuli özü (aşiq)
```

## Sufi istiarə sistemi
Klassik sufi şeirində istiarə **çoxqatlı məna sistem**dir:
- **Real məna** (zahir) — söz birinci anlam ilə
- **Mistik məna** (batın) — söz ilahi anlamla

### Sufi istiarələri
- **Şərab** = Allahın eşqi (mərifət)
- **Sevgili** = Allah
- **Meyxana** = sufi tərəfdarlıq mərkəzi
- **Saqi** = Allah-rəhbər
- **Gül** = ilahi gözəllik
- **Bülbül** = arif ruh

### Misal
```
"Sevgilimin yanağına şərab içirəm" 
- Zahir: real eşq haqqında 
- Batın: ilahi mərifəti dadıram
```

## İstiarənin gücü
- **Çoxqatlı məna** — bir söz çoxlu anlam daşıyır
- **Sıxılmış obraz** — kontekstdən oxucu özü anlamalıdır
- **Şəxsi sufi simvolizmi** — klassik divan poeziyasının yumşuq ipliyi
- **Sənət iddiası** — istiarə yaza bilmək sənətkarlıq nümayişi
- **Sənətkarlıq sınağı** — yanlış istifadə edilərsə oxucu anlamır

## Müasir istifadə

### Modern Azərbaycan şeirində
- Aqşin Yenisey: "**Söz** — bir bıçaq" (istiarə: söz = silah)
- Ramiz Rövşən: "**Şəhər** — köhnə kitab" (istiarə: şəhər = oxunan kitab)

### Pop və rap-da
- HOST punchline-larında istiarə tez-tez:
  - "**Mikrofon** bir tüfəng kimi" (istiarə + təşbeh)
- Modern pop chorus-larında:
  - "Sənsən **mənim qülləm**" (istiarə: sevgili = qüllə = möhkəmlik)

## İstiarə və metaforanın incə fərqi
**Sual**: "Ürəyim **dəniz**dir, sənsiz boş qalır." — istiarə yoxsa metafora?

**Cavab**: 
- **Metafora** desək: "ürək = dəniz" - bütöv köçürmə
- **İstiarə** desək: yalnız "dəniz" qalsa, "ürək" atılsa: "**Dəniz** sənsiz boş qalır" - oxucu kontekstdən anlamalıdır ki, "dəniz" = ürək

İstiarə daha sıxılmışdır. Yalnız müşəbbeh-bih qalır.

## İstiarə yazma metodu
1. **Mövzu seç** — istiarə üçün uyğun mücərrəd və ya konkret obyekt
2. **Müşəbbeh-bih tap** — obrazlı söz seçimi
3. **Müşəbbehı atın** — oxucu kontekstdən anlamalıdır
4. **Kontekst yarat** — oxucu yanlış anlamamalıdır
5. **Çoxqatlı məna nəzərdə tut** — istiarə bir mənalı olmasa daha güclüdür
6. **Sufi simvolizmi öyrən** — klassik istiarələr yaxşı materialdır

## Yanlışlar — qaçınmaq
- **Anlaşılmaz istiarə** (oxucu kontekst tapa bilmir)
- **Şablon istiarələr** (klassik gül-bülbül artıq köhnəlmiş bəzən)
- **Mənası açıq** istiarə (metafora olur)
- **Qarışıq simvol sistemləri** (sufi + modern, anlaşılmır)

## Klassik vs modern istiarə
| Klassik (sufi) | Modern |
|---|---|
| Şərab = mərifət | Çay = günəş açılması |
| Gül = sevgili | Pixel = xatirə |
| Bülbül = aşiq | Yağış = unutqanlıq |
| Şəm = sevgili | Ekran = uzaqlıq |
| Sərv = sevgili boyu | Göydələn = uzaqlıq |

## İnstrumentlərlə uyğunluq (istiarəli mahnılar üçün)
- **Tar + kamança + ud** (klassik sufi atmosfer)
- **Bütöv muğam ansamblı** (Alim Qasımov tərzi)
- **Solo piano** (modern istiarə üçün intim)
- **Ambient texture** (mistik istiarə üçün)
- **Spoken-word + minimal** (modern istiarə yansıdır)
