# Təşbeh və Metafora — Şeir Sənətinin Əsas Silahları

> Təşbeh — açıq müqayisə ("kimi", "tək", "misalı"). Metafora — gizli müqayisə (köçürmə). Klassik və modern Azərbaycan poeziyasının əsasları.

## Təşbeh (açıq müqayisə)
**Təşbeh** — bir şeyi başqa bir şeyə açıq müqayisə etməkdir. Əlamətli sözlər: **kimi, tək, misalı, sayağı, oxşar, bənzər**.

## Təşbehin əsas elementləri
1. **Müşəbbeh** — müqayisə edilən (mövzu)
2. **Müşəbbeh-bih** — müqayisə edilən obraz
3. **Vəc-i şəbəh** — müqayisə nöqtəsi (oxşarlıq)
4. **Ədat** — "kimi", "tək", "misalı"

### Misal
"Yarımın gözləri **göz yaşı kimi** parıldayır."
- Müşəbbeh: gözlər
- Müşəbbeh-bih: göz yaşı
- Vəc-i şəbəh: parıltı
- Ədat: kimi

## Təşbeh növləri

### 1. Mütlaq təşbeh (4 element tam)
"Yarımın yanağı **gül kimi qızarıb** açılıb."

### 2. Müjmel təşbeh (vəc-i şəbəh atılır)
"Yarımın yanağı **gül kimi**." (Niyə oxşar olduğu açıqlanmır)

### 3. Müəkkəd təşbeh (ədat atılır)
"Yarımın yanağı **gül**." (Birbaşa gül adı verilir, lakin metafora deyil çünki səs ilə təşbeh deməkdir)

### 4. Bəliğ təşbeh (3 elementsiz, ədat və vəc atılır)
"**Yarım — gül**." (Maksimum sıxılmış)

### 5. Müşahidə təşbehi
Real obyektin obrazlı təsviri: "Buludlar **pambıq kimi** göydə üzür."

## Klassik təşbehlər (divan ədəbiyyatı)

### Klassik müşəbbeh-bihlər
- **Yarımın gözü → ahu** (ceyran)
- **Yarımın saçı → şəbi-yelda** (uzun gecə)
- **Yarımın boyu → sərv** (sərv ağacı)
- **Yarımın qaşı → kaman** (yay-ox)
- **Yarımın dodağı → ləl, qönçə** (ləl daşı, gül qönçəsi)
- **Yarımın üzü → ay, gül, gün**
- **Yarımın yanağı → al gül**

### Klassik aşıq təşbehləri
- **Aşiq → bülbül** (gülə vurğun bülbül)
- **Yar → gül**
- **Hicran → cəfa, oda yanmaq**
- **Sevda → atəş, dəniz**

## Metafora (köçürmə) — gizli müqayisə
**Metafora** — bir sözün öz mənasından digər mənaya köçürülməsi. Müqayisə var, lakin **gizli**dir, "kimi" sözü işlədilmir.

### Misal
- Təşbeh: "Yarım **gül kimi** açıldı."
- Metafora: "Yarım **açıldı**." (gül müqayisəsi gizli, sadəcə "açılmaq" feli istifadə olunur)

### Klassik metaforalar
- "Ürəyim yandı" — eşqdən əzab çəkmək (atəş metaforası)
- "Gözümün nuru" — sevgili (görmə qabiliyyəti = sevgili)
- "Bağrım qana döndü" — şiddətli kədər (qan metaforası)
- "Söz qılıncdır" — söz yara verə bilər (qılınc metaforası)

## Metaforanın gücü
- **Sıxılmış məna** — bir sözə çoxqatlı məna yüklənir
- **Şeir incəliyi** — birbaşa demək yerinə obrazla göstərmək
- **Yaddaqalan obrazlar** — metafora yaddaşda iz qoyur
- **Sufi simvolizm** — klassik divan ədəbiyyatında metafora ilahi mənanı gizlədir

## Müasir təşbeh və metafora

### Modern Azərbaycan şeirində
- **Ramiz Rövşən** — şəhər təşbehləri ustası
  - "Şəhər **bir köhnə kitab kimi**, hər küçəsi səhifə."
- **Aqşin Yenisey** — minimalist metafora ustası
  - "Söz — bir bıçaq."
- **Salam Sarvan** — cəsarətli metafora
  - "Tabular insan üçün dəri kimidir."

### Modern pop və rap-da
- Pop chorus-da klassik təşbehlər ("ürəyim **kimi** döyünür")
- Rap-da modern təşbehlər ("bu sistem **bir qəfəs kimi**")
- HOST punchline-larında metafora ustadlığı

## Misal: Təşbeh + metafora birgə (modern lirik)
```
Sənin gözlərin (müşəbbeh)
qaranlıq dənizdir (metafora — birbaşa "dəniz" deyilir, "kimi" yox),
mənim sərbəst hərəkətim
o sularda boğulur (metafora — "boğulmaq" eşqdə çıxılmaz olmaq).
```

## Təşbehin hipotetik və real növləri

### Real təşbeh
"Yarımın gözləri **dəniz kimi** maviydir." (göz rəngi həqiqətən mavi olduğunda)

### Hipotetik təşbeh
"Yarımın gözləri **dəniz kimi** dərindir." (real ölçülməz, hisslə demək)

### Mübaliğəli təşbeh
"Yarımın gözəlliyi **mələk kimi**." (mübaliğə var)

## Modern istifadə müsbət misallar

### Pop ballada
"Sənin gəlişin **bahar kimi**, hər tərəf çiçək açar."

### İndie folk
"Bu küçələr **xatirələr kitabı kimi**, hər səhifə bir gün, hər gün bir yara."

### Rap
"Söz **bıçaq kimi**, kəsər həm ürək həm kəlam." (HOST tərzi)

## Təşbeh və metafora yazma metodu
1. **Mövzu seç** — nə təsvir etmək istəyirsən?
2. **Real təsvir** yaz — tam obyektiv
3. **Müşəbbeh-bih axtar** — müqayisə üçün uyğun obraz
4. **Vəc-i şəbəh** tap — oxşarlıq nədədir?
5. **Təşbeh formula** "kimi/tək/misalı" qoy
6. **Metaforaya çevirmək üçün** — "kimi"-ni at, birbaşa de
7. **Yenidən oxu** — obraz əslindən güclüdürsə, uğurludur

## Yanlışlar — qaçınmaq
- **Şablon təşbehlər** ("göz = ay" milyon dəfə işlənib, orijinal yox)
- **Uzun-uzadı təşbeh** (sıxılmış olmalıdır)
- **Əlaqəsiz müqayisə** (göz ilə banan müqayisə etmək — uyğun yox)
- **Liyalı metafora** (mənası anlaşılmırsa pozulur)
- **Modern şablonlar** ("bayraq kimi", "dəniz kimi" yorulmuş)

## Klassik vs modern təşbehlər
| Klassik | Modern |
|---|---|
| Göz = ahu | Göz = ekran piksellər |
| Saç = şəbi-yelda | Saç = qaranlıq metro tuneli |
| Boy = sərv | Boy = göydələn |
| Sevda = atəş | Sevda = elektrik şoku |
| Yar = gül | Yar = playlist hər mahnısı |

## Təşbeh və metaforanın aşıq sənətindəki yeri
Aşıq Ələsgər və başqa aşıqlar təşbeh ustaları idi. Bədahətən söz deyərkən sürətlə klassik müşəbbeh-bihlər seçirdilər. Modern aşıq da bu disiplini saxlayır.
