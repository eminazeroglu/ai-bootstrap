# Alliterasya — Səs Təkrarı və Musiqi Effekti

> Eyni səsin (samit və ya sait) təkrarlanması ilə şeirə musiqi gətirmə. Şeirin "səslənən" tərəfi.

## Tərif
Alliterasya — şeirdə **eyni səsin (samit və ya sait) bir misrada və ya yan-yana sözlərdə təkrarlanmasıdır**. Şeirə daxili musiqi və axıcılıq verir, oxunaqlığını artırır, yaddaqalan edir.

## Alliterasya növləri

### 1. Samit alliterasyası
Eyni samit səsin təkrarı.

#### "S" alliterasyası (yumşaq, lirik):
```
"Sevda saçlı sevgilim, sənsən sirrim, sənsən söz."
```

#### "K" alliterasyası (sərt, kəskin):
```
"Küçələr küləyi, kəskin kəlamlar, kim qabaqlayar?"
```

#### "M" alliterasyası (yumşaq, dolğun):
```
"Mənim mehribanım, mənim məhbubum, mənim mahalım..."
```

#### "B" alliterasyası (dolğun, dramatik):
```
"Bahar bizim bağa boylu boylu bənövşə bəzər..."
```

### 2. Sait alliterasyası (assonans)
Eyni saitin təkrarı.

#### "A" assonansı (açıq, geniş):
```
"Ana, ana, ananın atası ay anar."
```

#### "Ə" assonansı (yumşaq, intim):
```
"Mənə, sənə, hər nəfəsə əzizdir əbədi əfsanə."
```

#### "İ" assonansı (kəskin, parlaq):
```
"İnci, irmaq, izi izi şirin, sirli, içi içi gülşən."
```

### 3. Onomatopoeia (səs təqlidi alliterasya ilə)
Səs verən söz alliterasyası.
```
"Şır-şır su şəlalə, şıltaq şehlər şəfəq."
```

### 4. İçəri qafiyə (assonans + rhyme)
Misra daxilində səs təkrarı.
```
"Sevda söndü, könül donaldı, can kül oldu, qan qaynadı."
```

## Klassik alliterasya nümunələri

### Klassik Azərbaycan şeirində
Aşıqlar bilərəkdən alliterasya istifadə edirdilər çünki saz müşayiətində səs uyğunluğu vacibdir.

#### Aşıq Ələsgər tərzi:
```
"Yarpaq yenildi, yaz yetişdi, yenidən yad düşdü yarım yenə də..."
(Y səsi alliterasyası — axıcı, lirik)
```

### Klassik divan ədəbiyyatında
Füzuli, Nəsimi alliterasya master-ləri.

#### Füzuli (uyğunlaşdırma):
```
"Sevda söndü, sən söylə sözünü, sənsiz sədəm sönər saraylar."
```

## Modern alliterasya nümunələri

### Modern lirik şeirdə
Aqşin Yenisey, Salam Sarvan, Ramiz Rövşən alliterasya istifadə edirlər kəskin və ya yumşaq effekt üçün.

#### Aqşin Yenisey (kəskin):
```
"Söz — saralmış, sönmüş, susmuş, salam-əleyküm."
```

#### Ramiz Rövşən (yumşaq):
```
"Şəhər şüşəsi, şah şüşəsi, şirin şüşə, şikəst şüşə."
```

### Modern pop və rap-da

#### Pop ballada (yumşaq alliterasya):
```
"Sənə sözüm, sirrim, sevdam, sənsən mənim hər şeyim."
```

#### HOST tərzi rap (kəskin alliterasya + cinas):
```
"Söz savaşı, söz silah, sözüm sənə sed-saxla,
Sənət sözüm, sənət sözə söykənir, sözə yan baxma."
```

## Alliterasyanın gücü
- **Daxili musiqi** — şeir öz-özlüyündə musiqili olur
- **Yaddaqalanlıq** — alliterasyalı sətir yaddaşa möhkəm yatır
- **Emosional ton** — səslər emosiyaya təsir edir (S = yumşaq, K = sərt)
- **Saz/musiqi uyğunluğu** — alliterasyalı söz musiqilə daha yaxşı uyğunlaşır
- **Sənət incəliyi** — alliterasya ustası şair texnikası nümayiş etdirir

## Səslərin emosional yükü

| Səs | Emosional ton | Uyğun mövzu |
|-----|---------------|--------------|
| S, ş, z | Yumşaq, intim, sirli | Sevgi, sirr, layla |
| K, q, x | Sərt, kəskin, dramatik | Mübarizə, qəzəb |
| M, n | Dolğun, mehriban | Layla, ana sevgi |
| B, p | Dolu, qüvvətli | Bahar, qüdrət |
| L, r | Axıcı, sürətli | Hərəkət, rəqs |
| A, ə | Açıq, geniş | Vətən, fəlsəfə |
| İ, e | Kəskin, parlaq | Sevinc, parlaqlıq |
| O, u | Dolğun, dərin | Kədər, dərinlik |

## Modern istifadə

### Pop hook (S alliterasyası)
```
"Sənsən, sənsən, sənsən mənim sevdam,
Sirrim sənsən, sözüm sənsən, sevda sənsən."
```

### Rap punchline (K alliterasyası)
```
"Kim qaçar, kim qabaqlar, kim qalxar, kim qonqar?
Kələk-koladan kompromis, kompozisya, kompozit."
```

### Folk-pop ballada (Y alliterasyası)
```
"Yarımı yad eylədi yağış, yorğun yarpaq yenildi,
Yaz yetişdi, yarım yenidən yad oldu yenə."
```

## Alliterasyanın incəliyi
Alliterasya **bilərəkdən** edilməlidir. Təsadüfi səs təkrarı yox, **bilərəkdən düzülmüş** səs sistemi.

### Yaxşı alliterasya
```
"Sevda söndü, sən sözünü söyləməyi unutdun, mən susdum səssiz."
```
(S səsi mənaca uyğun mövzuda — sevgi, sirr, sükut)

### Pis alliterasya
```
"Sevda söndü, sən kitabı oxudun, mən susdum səhifə açdım."
```
(S səsi var, lakin mənaca uyğunsuz qarışıq)

## Alliterasya yazma metodu
1. **Mövzu seç** — alliterasya hansı emosiyanı dəstəkləyəcək?
2. **Səs seç** — mövzuya uyğun səs tap (S = sevgi, K = mübarizə)
3. **Söz axtar** — həmin səslə başlayan və ya çoxlu olan sözlər
4. **Misra qur** — sözlər təbii axsın, zorlanmış olmasın
5. **Yenidən oxu** — alliterasya hiss olunsun, lakin uyğunsuz görünməsin
6. **Mənanı qoru** — alliterasya məna xərcinə olmasın

## Yanlışlar — qaçınmaq
- **Zorlanmış alliterasya** (sözlər uydurulmuş kimi səslənir)
- **Mənaca uyğunsuz** alliterasya (səs uyğun, mənaca uyğunsuz)
- **Çox alliterasya** (bütün şeir sıxıcı olur)
- **Yanlış səs seçimi** (kədərli mövzuda L alliterasyası uyğun deyil)

## Alliterasyanın aşıq sənətində xüsusi yeri
Aşıq sənətində alliterasya saz müşayiəti üçün xüsusi vacibdir:
- **Saz tellərinin səsi** ilə alliterasyalı sözlər uyğunlaşır
- **Bədahətən söz** deyərkən aşıq alliterasyalı sözlər seçə bilər
- **Yaddaqalanlıq** — alliterasyalı qoşma daha yaxşı yadda qalır
