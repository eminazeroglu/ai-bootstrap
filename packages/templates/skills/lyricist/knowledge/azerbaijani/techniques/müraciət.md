# Müraciət — Şeirdə Xitab Sənəti

> Şair birbaşa kimsəyə (sevgili, vətən, Allah, oxucu, özünə) müraciət edir. Klassik divan ədəbiyyatı və modern lirikanın əsas texnikası.

## Tərif
Müraciət — şairin şeirdə **birbaşa kimsəyə xitab etməsidir**. Mücərrəd və ya konkret obyektə (sevgiliyə, vətənə, ruhuna, oxucuya, Allaha, ölümə) "sən" formasında müraciət şeirə dramatik dinamika gətirir.

## Müraciətin əsas elementləri
1. **Xitab edən** — şair (mən)
2. **Xitab edilən** — sevgili, vətən, Allah, və ya başqa obyekt
3. **Xitab forması** — "Sən..., ey..., ay..., ey gözəl..."
4. **Mövzu** — niyə müraciət olunur?

## Müraciət növləri (xitab edilənə görə)

### 1. Sevgiliyə müraciət
Klassik və modern lirikada ən geniş.
```
"Ey sevgilim, mənim ürəyim sənin əlindədir,
Sən demə bu sevda yalandı..."
```

### 2. Vətənə müraciət
Vətəndaşlıq mövzulu şeirlərdə.
```
"Ey vətən, sənsiz yaşaya bilmərəm,
Sənin torpağı altında bir gün yatacam..."
```

### 3. Allaha müraciət (sufi və klassik)
İlahi xitab.
```
"Ey Rəbb, sənsiz dünya boş, mən boş, hər şey boş,
Sənin nuruna pənah, sənsən mənim hər şeyim..."
```

### 4. Ölümə müraciət
Ekzistensial və klassik şeirlərdə.
```
"Ey ölüm, gəlsən gəl, qorxmuram artıq,
Mən səndən əvvəl çoxdandır ölmüşəm..."
```

### 5. Şəxsi obyektə müraciət (klassik)
"Ey könül", "ey ürək", "ey can" — özünə müraciət.
```
"Ey könül, niyə yenə də sevda yoluna düşdün?
Bilmirsənmi sevda dərd gətirir..."
```

### 6. Təbiətə müraciət
"Ey gül", "ey bülbül", "ey dəniz" — təbiət obyektləri ilə dialog.
```
"Ey gül, sən niyə bu qədər tez soldun?
Sənin yarpaqların kimi, mənim ümidlərim də soldu..."
```

### 7. Cəmiyyətə / oxucuya müraciət (modern)
"Ey insanlar", "ey gənclər", "ey siz oxuyanlar".
```
"Ey insanlar, eşidin sözümü,
Bu dünya yalandır, bilirsiniz?"
```

### 8. Ölmüşə müraciət
Yaslı şeirlərdə.
```
"Ey rəhmətlik, sən bizdən ayrı düşdün,
Lakin xatirən qaldı bizimlə əbədi..."
```

### 9. Soyut məfhumlara müraciət
"Ey vaxt", "ey həyat", "ey sevda".
```
"Ey vaxt, niyə bu qədər tez keçirsən?
Mən hələ yaşamağa başlamamışam..."
```

## Müraciət forma sözləri (klassik)

### Klassik xitab sözləri
- **Ey** (ən geniş): "Ey gözəl", "Ey vətən", "Ey könül"
- **Ay** (lirik): "Ay yarım", "Ay sevgilim"
- **Ya** (klassik divan): "Ya sevgilim", "Ya Rəbb"

### Modern xitab sözləri
- **Sən** (birbaşa): "Sən, mənim sevgilim..."
- **Hey** (modern, gənc): "Hey, sevgilim..."
- **Sevgilim** (birbaşa): "Sevgilim, sənə demək istəyirəm..."

## Klassik müraciət misalları

### Füzuli (sevgiliyə)
```
"Ey sevgilim, mənim qəlbimi yandırdın,
Sənsiz necə yaşaya bilərəm bu dünyada?"
```

### Aşıq Ələsgər (vətənə)
```
"Ey doğma vətən, sənsən hər nəfəsim,
Sənin torpağın mənim canım, qanım..."
```

### Nəsimi (Allaha — sufi xitab)
```
"Ya Rəbb, mən sənin nurundayam,
Sənsiz mən yox, sən mənsiz yox..."
```

## Modern müraciət misalları

### Səməd Vurğun (vətənə)
```
"Ey vətən, mən sənin oğlunam,
Sənin torpağı mənim qanım, canım..."
```

### Aqşin Yenisey (özünə)
```
"Ey könül, neçə dəfə dedim sənə —
Sevda yolu yox, qızı yox, yox heç nə..."
```

### Ramiz Rövşən (şəhərə — modern)
```
"Ey Bakı, sənin küçələrin mənim damarlarım,
Sənin gecələrin mənim yuxularım..."
```

## Müraciətin gücü
- **Dramatik dinamika** — şeirə "dialog" hissi gətirir
- **İntimacy** — şair və xitab edilən arasında bağ qurur
- **Emosional vurğu** — birbaşa "sən" formul daha kəskin
- **Rhetorical effect** — oxucu da xitab edilən rolunda hiss edir
- **Klassik miras** — divan ədəbiyyatından müasir şeirə qədər

## Müraciət + təşbeh + metafora birləşməsi
```
"Ey sevgilim (müraciət), sən mənim **ulduzum**sansan (metafora),
Hər gecə **işıq saçan** parlağım (təşbeh)..."
```

## Modern istifadə

### Pop mahnılarda
Müraciət modern pop mahnıların əsasındadır:
```
[Verse 1]
Sən mənim hər nəfəsim,
Sən mənim hər kəlamım...

[Chorus]
Sevgilim, qal mənimlə,
Sevgilim, getmə yenidən...
```

### Rap-da müraciət (HOST tərzi)
```
[Verse]
Ey gənclər, eşidin sözümü,
Bu dünya yalandır, bilirsiniz?
Ey rəfiqlər, açın gözünüzü,
Söz qılıncdır, hər söz bir yara..."
```

### Modern indie-rock müraciət
```
"Sən, mənim şəhərim,
Sən, mənim küçələrim,
Sən, mənim hər addımım..."
```

## Müraciət yazma metodu
1. **Xitab edilənı seç** — sevgili, vətən, Allah, özün, təbiət, ölüm
2. **Xitab forması seç** — "Ey", "Ay", "Sən", modern və ya klassik
3. **Mövzunu təyin et** — niyə müraciət edirsən?
4. **Direkt "sən" formul** — birinci və ikinci şəxs dialoq
5. **Emosional ton** — xitab edilənlə əlaqə nədir? (sevgi, hörmət, qəzəb, kədər)
6. **Şeir boyu davam etdir** — müraciət bütün mətndə qalsın

## Yanlışlar — qaçınmaq
- **Müraciət başlasın, sonra dəyişsin** (forma birliyi pozulur)
- **Çox formal müraciət** modern mahnılarda (sıxıcı olur)
- **Müraciət edilənin natamam tanıdılması** (oxucu kimə xitab olunduğunu anlamamalıdır)
- **Şablon müraciətlər** ("ey gözəl" yorulmuş)

## Müraciətin "yaşaması" niyə vacibdir?
- **Lirikanın əsas özəlliyi** — müraciətsiz lirika yarımdır
- **Modern mahnı strukturu** — verse + chorus müraciət əsasdır
- **Klassik miras** — divan ədəbiyyatından modern poeziyaya qədər
- **Dialogue effect** — şeir bir tərəfli monolog deyil, dialoq olur

## Müraciətin mövzu seçiminə təsiri
- **Sevgiliyə müraciət** → eşq, romantik, lirik mahnı
- **Vətənə müraciət** → vətəndaşlıq, milli ruh
- **Allaha müraciət** → sufi, mistik, dini
- **Özünə müraciət** → fəlsəfi, ekzistensial
- **Şəhərə müraciət** → modern urban lirika
- **Ölümə müraciət** → ekzistensial, dramatic
