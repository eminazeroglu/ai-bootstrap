# Azerbaijani Rock / Indie Mahnı Sözləri

> Azerbaijani rock musiqisi nadir, lakin canlıdır. Bulvardan, Mənsuriyə, FREE — modern indie. Sərbəst vəzn dominant.

## Tərif
Azerbaijani rock — XX əsr sonu və XXI əsrdə formalaşmış müasir rock janrıdır. Pop musiqisi qədər yayqın deyil, lakin **indie**, **alternative rock**, **folk-rock**, və **post-rock** versiyalarında canlı qrupları var. Sözlər çox vaxt **sərbəst vəzndə**, Qərb rock estetikası + Azərbaycan dili sintezində.

## Tarixi inkişaf

### Mərhələ 1: Erkən rock cəhdləri (1970-90)
- Sovet dövründə rock yarımyasaq idi
- Bəzi underground qruplar
- Əsasən rus dilində ifa
- Azərbaycan dilində rock nadir

### Mərhələ 2: Müstəqillik dövrü (1990-2000)
- İlk Azərbaycan dilli rock qrupları
- Punk, alternative təsiri
- Underground hərəkat

### Mərhələ 3: Modern dövr (2000+)
- **Bulvardan** — folk-rock fusion
- **Mənsuriyə** — alternative rock
- **FREE** — modern indie-rock
- **AzRock** hərəkatı yaranır
- Festival mədəniyyəti (Karvan, Caspian Crossroads)

### Mərhələ 4: Yeni nəsil (2015+)
- Bedroom indie produksiyası
- DIY culture
- Spotify, YouTube üçün yazılır
- Dinləyici kütlə kiçik, lakin sədaqətli

## Azerbaijani rock-un xüsusiyyətləri

### 1. Sərbəst vəzn dominant
- Klassik aşıq vəzni rock üçün uyğun deyil
- Modern sərbəst şeir + rock musiqisi uyğun
- Söz axarı təbii, danışıq diline yaxın

### 2. Şəxsi / ekzistensial mövzular
- "Mən kimsəm?" tipli sual
- Şəhər həyatı, depressiya, anxiety
- Modern alienation

### 3. Sosial-tənqidi mövzular (cəsarətlə)
- Rock janrı tarixən üsyankar
- Sistem tənqidi
- Cəmiyyət problemləri

### 4. Folk-rock fusion
- Aşıq sənəti + rock instrumentlər
- Saz + elektron gitara
- Klassik mövzu + modern aranjman

### 5. İngilis dili qarışıqlığı (bəzən)
- Modern indie-rock-da
- Bilingual sözlər
- "Code-switching" estetikası

## Mövzular

### 1. Ekzistensial / fəlsəfi
- Həyatın mənası
- Yalnızlıq
- Depressiya / mental sağlamlıq
- Modern indie-rock dominant

### 2. Sosial / siyasi
- Cəmiyyət tənqidi
- Sistem nezamı
- Fərd və cəmiyyət
- Punk / alternative tərzi

### 3. Şəxsi münasibətlər
- Eşq, lakin daha kompleks (toxic, broken)
- Aile dramaları
- Dostluq

### 4. Şəhər həyatı
- Bakı urban realism
- Modern yorğunluq
- Şəhər izolyasiyası

### 5. Vətən / kimlik (alternativ)
- Vətəndaşlıq sual altında
- Diaspora mövzusu
- Modern Azərbaycan kimliği

### 6. Sənət və yaradıcılıq (meta)
- Sənət haqqında sənət
- Rock kültürü
- Underground manifestoları

## Rock struktur

### Standart rock mahnı (3-5 dəqiqəlik)
```
[Intro] — gitara riff
[Verse 1] — sərbəst vəzn, hekayə açılır
[Chorus] — yaddaqalan, daha dramatic
[Verse 2] — yeni hekayə
[Chorus] — eyni
[Bridge / Solo] — gitara solo və ya emosional pik
[Final Chorus] — daha böyük
[Outro] — fade və ya kəskin sonluq
```

### Indie / atmospheric rock
- Daha uzun (5-8 dəqiqə)
- Verse uzun, hekayə-yönlü
- Chorus daha minimal
- Bridge dramatic kataktik

### Punk / hardcore
- Qısa (2-3 dəqiqə)
- Sürətli (BPM 140+)
- Chorus aggressive
- Mövzu cəsarətli

## Misal: Modern indie-rock (Bulvardan tərzi)
```
[Intro — fingerpicked acoustic guitar]

[Verse 1 — sərbəst, intim]
Şəhər yatır, mən oyaq qalmışam yenə
Pəncərədən baxıram, çıraqlar danışır
Sənsiz hər şey başqa cür
gözəldi, başqa cür ağır

[Pre-Chorus — gərginlik artır]
Bilirəm, gec olmuşdur,
amma yenə də deyirəm sənə...

[Chorus — dramatic, hook]
Qal mənimlə bir də gecə,
İcazə ver hər şeyi unutmağa,
Qal mənimlə bir də gecə,
Sabaha qədər biz olmağa.

[Verse 2]
...
```

## Misal: Alternative rock (Mənsuriyə tərzi)
```
[Verse 1 — sərbəst, sosial]
Bu sistem mənə qadağan deyir,
Hər addımım təftiş altındadır,
Mən susmuram, lakin söz acıdır,
Bu nə dünyadır, kim cavab verə?

[Chorus — aggressive, dramatic]
Söz susmaz!
Söz susmaz!
Hər söz bir güllə, hər nəfəs bir döyüş,
Söz susmaz!
```

## Misal: Folk-rock fusion
```
[Verse — sərbəst, lirik]
Doğma vətən, sənsən ürəyim,
Hər nəfəsim sənə qonsun,
Sənin torpağı altında bir gün
mən də səssiz yatacam.

[Chorus — heca vəznli, gəraylı sayağı]
Vətənim, vətənim, vətənim,
Hər nəfəs sənə bağlı,
Sənsiz mən yox, sən yox mənsiz,
Bir-birimizə bağlıyıq biz.
```

## Rock yazma metodu
1. **Mövzu seç** — rock cəsarətli mövzu daşıyır (ekzistensial, sosial, kompleks eşq)
2. **Verse-də sərbəst vəzn** — hekayə təbii axsın
3. **Chorus üçün heca vəznli hook** (8 hecalı yaxşıdır)
4. **Bridge dramatic kataktik** — gitara solo və ya emosional pik
5. **Söz cəsarəti** — pop-un sərhəddini aş
6. **Folk izi** (folk-rock üçün) — saz və ya tar accent
7. **Modern indie production** — lo-fi və ya polished modern

## Yanlışlar — qaçınmaq
- Pop standartları (rock öz disipliniylə yazılır)
- Çox bayağı eşq mövzusu (rock kompleks mövzu istəyir)
- Hook-suz chorus (modern rock-da hook lazımdır)
- Klassik divan dili (rock müasir dil tələb edir)
- Türk rock təqlidi (Azərbaycan rock-u öz səsini tapmalıdır)

## Folk-rock — Azərbaycan rock-unun ən orijinal forması
Bulvardan və oxşar qrupların folk-rock fusion-u Azərbaycan rock-unun ən orijinal estetikasıdır:
- Aşıq sənəti təsiri
- Saz + elektron gitara
- Klassik mövzu + modern aranjman
- Vətən / klassik şeir uyğunlaşdırması
