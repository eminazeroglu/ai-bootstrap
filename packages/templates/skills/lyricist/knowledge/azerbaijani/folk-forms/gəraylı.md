# Gəraylı — Aşıq Sənətində Lirik Forma

> 8 hecalı, axıcı aşıq forması. Saz müşayiəti ilə oxunur. Lirik və yumşaq.

## Folk-form perspektivindən
Bu fayl gəraylını **aşıq sənəti** baxımından izah edir. Vəzn perspektivindən geniş baxış: `meter/8-heca-gəraylı.md`.

## Aşıq gəraylısının xüsusiyyətləri
- **8 hecalı, 4+4 duraq** — axıcı və oxunaqlı
- **4 misralı bənd** — qoşmadan az misralı (qoşma 11 hecalı)
- **aaab + cccb** — rədifli forma
- **Saz müşayiəti** — qoşmadan daha yumşaq melodik xət
- **Lirik mövzular dominant** — eşq, gözəllik, təbiət

## Aşıq gəraylısının janrları

### 1. Sevgi gəraylısı (ən geniş)
- Sevgili gözəlliyi
- Lirik təsvir
- Yumşaq, mehriban ton

### 2. Təbiət gəraylısı
- Bahar, yaz fəsli, dağlar
- Doğa təsviri
- Aşıq Şəmşir master

### 3. Vətən gəraylısı
- Doğma yer
- Lirik vətən təsviri
- Müasir aşıqlar geniş istifadə edir

### 4. Layla / oxşama tərzi
- Beşik mahnılarına yaxın
- Yumşaq, sakit
- Ana laylaları gəraylı vəznli olur tez-tez

## Quruluş
```
Bənd 1 (giriş):
aaab — 4 misra, 8 heca hər biri
4+4 duraq

Bənd 2-5 (inkişaf):
cccb, dddb, eeeb...
b — rədif (təkrarlanan son misra)

Son bənd:
Aşıq adı (təxəllüs) çəkilir
Mövzu yekunlaşdırılır
```

## Misallar (Aşıq Ələsgər tərzi)

### Sevgi gəraylısı
```
Səhər oldu, oyandım mən,
Pəncərədən baxdım gülümsəyən,
Hər tərəfi nur, ay çiçəyim,
Yarımı gözüm görmədi.

Düzlər yaşıllıq geyindi,
Buludlar göydən endi,
Hər tərəf nur, ay sevgilim,
Yarımı gözüm görmədi.

(...)

Ələsgər deyir: hələ də yox,
Səbrim tükənib, dilim bağlandı sox,
Bir əlin uzat, ay çiçəyim, çox,
Yarımı gözüm görmədi.
```

### Təbiət gəraylısı (Aşıq Şəmşir tərzi)
```
Bahar gəldi, yaz çiçəkli,
Dağlar yaşıl, çöl bəzəkli,
Bülbül oxur gül üzərində,
Yer üzü cənnət olub yenə.

(...)
```

## Aşıq gəraylısının ifa tərzi
- **Saz açılışı yumşaq** (qoşmadan daha melodik)
- **Səs intonasiyası lirik** (qoşmadan daha sakit)
- **Bədahətən söz** (improvizasiya əsas)
- **Yumşaq melodik xət** — gəraylı dramatic deyil, yumuşaqdır
- **Beşik mahnısına yaxınlıq** — yatızdırma intonasiyası

## Tanınmış aşıq gəraylı ustaları
- **Aşıq Ələsgər** — gəraylı + qoşma + müxəmməs üçlü ustad
- **Aşıq Şəmşir** — təbiət gəraylısı master
- **Aşıq Pənah** — müasir
- **Aşıq Mirzə** — qadın aşıq tradisiyasında
- **Aşıq Mikayıl Azaflı** — tanınmış müasir aşıq

## Müasir uyğunlaşdırma

### Folk-pop
- 8 hecalı verse + chorus axıcılığı modern folk-pop üçün ideal
- Brilliant, Aygün Kazımova, Manaf Ağayev gəraylı strukturlu mahnılar oxuyur

### Lullaby (layla)
- Beşik mahnıları çox vaxt gəraylı vəznlidir
- Yumşaq, ritmik, yatızdırıcı

### Rap-da gəraylı izləri
8 hecalı qısa, axıcı misralar rap chorus üçün uyğundur. HOST grupu bəzi mahnılarda gəraylı strukturuna yaxın chorus istifadə edir.

## Qoşma vs Gəraylı (aşıq sənətində)
| | Qoşma | Gəraylı |
|---|---|---|
| Heca | 11 | 8 |
| Duraq | 6+5 | 4+4 |
| Hava | Geniş, dramatic | Yumşaq, lirik |
| Mövzu | Geniş spektr | Lirik dominant |
| Saz xətti | Daha dramatic | Daha melodik |
| İstifadə | Dominant | Lirik mövzularda |

## Aşıq gəraylısı yazma metodu
1. **Lirik mövzu seç** — yumşaq, mehriban (sevgi, təbiət, layla)
2. **8 heca + 4+4 duraq** sabit qal
3. **aaab birinci bənddə** (rədif b)
4. **Sonrakı bəndlərdə cccb** struktur
5. **Yumşaq dil seçimi** — gəraylı sadə, axıcı
6. **Son bəndcontextRoot təxəllüs** — aşıq imza atmalıdır
7. **Saz açılışı təsəvvür et** — gəraylının melodik xətti yumşaq olmalı

## Yanlışlar — qaçınmaq
- 7 və ya 9 heca işlətmə
- 4+4 duraqdan kənara çıxma
- Dramatic mövzu (gəraylı lirik qalmalı)
- Rədifi pozmaq
- Təxəllüsü unutmaq

## Aşıq gəraylısının canlılığı
- **Layla mirası** — Azərbaycan analar gəraylı vəznli laylalar oxuyub uşağı yatırdır
- **Yumşaq atmosfer** — toy mərasimləri, sevgi söhbətləri üçün uyğun
- **Modern folk-pop** — bugünki Azərbaycan folk-pop musiqisinin əsasında dayanır
- **Kommersiya uğuru** — 8 hecalı qısa misra Suno və başqa AI-da yaxşı işləyir

## İnstrumentlərlə uyğunluq
- **Saz** (məcburi əsas, gəraylı üçün daha yumşaq xət)
- **Balaban** (lirik melodik xət köməkçi)
- **Tar** (genişləndirmə)
- **Akustik gitara fingerpicking** (modern folk-pop uyğunlaşdırma)
- **Piano** (kamera versiyası)
- **String quartet** (chamber-folk fusion)
