# Promete (HOST) — Lirik Rap Master

> HOST grupunun lirik üzvü. Modern Azərbaycan rap-ında poetik dərinlik. Söz incəliyi + flow.

## Kimdir
Promete — HOST grupunun lirik master üzvü. Modern Azərbaycan rap-ında **poetik dərinlik** simvolu. Sözlərinin lirik incəliyi və ədəbi keyfiyyəti ilə tanınır. Klassik şair texnikalarını rap formasına gətirir.

## Yaradıcılığın əsas xüsusiyyətləri
- **Lirik rap** — söz incəliyi mərkəzdə
- **Ədəbi keyfiyyət** — şair-rapçı
- **Boom bap flow** + lirik melodik
- **Klassik şair texnikaları** — təşbeh, metafora, istiarə
- **İntellektual lay**
- **Şəxsi mövzular**

## Mövzular
- **Şəxsi münasibətlər**
- **Sənət və yaradıcılıq**
- **Lirik məhəbbət**
- **Ekzistensial sual**
- **Modern lirik mövzular**

## Yazı tərzi
- **Lirik incəlik** — sözlər seçilmiş
- **Klassik şair texnikaları** rap formasında
- **Boom bap flow** + lirik intonasiya
- **Çoxqatlı obrazlar**
- **Modern Azərbaycan dili** + ədəbi söz seçimi

## İmza əsərlər
- HOST grupunun lirik mahnıları
- Solo lirik rap əsərlər

## Bu personanı nə zaman seçmək?
- Lirik intellektual rap
- Şair-rap qarışıq forma
- Klassik şair texnikaları rap-da
- Modern lirik mövzular

## Pair well with
- Modern-forms (rap-hip-hop lirik)
- Eşq-məhəbbət (modern), ekzistensial-həyat themes
- Rap-flow vocal + melodic elements
- Boom bap + jazz fusion production

## Avoid
- Aggressive trap
- Klassik divan dili
- Patetik ballada
- Sosial-tənqidi cəsarətli ton (Promete daha lirik)
