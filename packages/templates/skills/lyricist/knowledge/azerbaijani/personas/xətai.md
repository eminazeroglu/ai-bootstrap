# Şah İsmayıl Xətai (1487-1524) — Hökmdar-Şair

> Səfəvilər dövlətinin qurucusu, eyni zamanda klassik divan şairi. Qızılbaş tərəfdarlığı və sufi şeiri.

## Kimdir
Şah İsmayıl Xətai — Səfəvilər sülalsinin qurucusu (1501-də şah), eyni zamanda **böyük sufi şair**. "Xətai" təxəllüslü. Klassik Azərbaycan dilində qəzəllər yazıb. Qızılbaş təriqətinin lideri. 22 yaşında "Şahnamə"ni Azərbaycan dilinə uyğunlaşdıran ilk şair.

## Yaradıcılığın əsas xüsusiyyətləri
- **Hökmdar-şair** — siyasi güc + sənət birgə
- **Azərbaycan dilində klassik divan** (fars dilinə alternativ)
- **Sufi (qızılbaş tərəfdarlığı) şeiri** — ələvi-şiə mövzular
- **Heydər (atası) ardıcıllığı**
- **Cəsarətli mənəvi mövzu**

## Mövzular
- **Qızılbaş tərəfdarlığı** — şiə-ələvi mövzu
- **Sufi mənəviyyat** — mənəvi yol
- **İlahi eşq**
- **Hökmdar əxlaqı** (bəzi şeirlərdə)
- **Vətən-tarix mövzusu**

## Yazı tərzi
- **Klassik Azərbaycan dili** dominant (fars az)
- **Sadə və dərin** birgə
- **Sufi simvolizmi** + tərəfdarlıq mövzusu qarışıq
- **Hörmətli, ləyaqətli ton** (hökmdar olduğu üçün)

## İmza misraları (uyğunlaşdırma)
```
Nə fariqəm, nə dolanım, mən bilirəm təriqi,
Sıdqı bütün sevirəm, mən mən özüməm bağrı.
```

```
Şahım Heydər, sənə qurban, sənin yolunda yandım,
Mənim canım sənindir, sən mənim hər şeyimsən.
```

## Modern istifadə
- Klassik aşıq və xanəndə repertuarında bəzi parçalar
- Tarixi-sufi mövzulu modern uyğunlaşdırmalar
- Səfəvilər dövrü mədəni miras kontekstində

## Bu personanı nə zaman seçmək?
- Klassik Azərbaycan divan ədəbiyyatı (fars yox)
- Qızılbaş / şiə-sufi mövzusu
- Hökmdar tonalitəsi (ali, ləyaqətli)
- Tarixi-mənəvi mövzular
