# İmadəddin Nəsimi (1369-1417) — Sufi Şeirinin Şəhidi

> "Ənəlhəq" (Mən Haqqam) deyən hürufi sufi şair. Vəhdət-vücud mövzusunun tac şairi. Halab şəhərində dərisi soyularaq edam edilib.

## Kimdir
İmadəddin Nəsimi — XIV-XV əsr klassik Azərbaycan sufi şairi. **Hürufilik** məzhəbinin nümayəndəsi. Azərbaycan dilində və fars dilində qəzəllər yazıb. "Ənəlhəq" (Mən Haqqam) ifadəsinə görə Halabda dərisi soyularaq edam edilib (1417). Sufi şəhidlərinin simvoludur.

## Yaradıcılığın əsas xüsusiyyətləri
- **Sufi (təsəvvüf) şair** — vəhdət-vücud doktrinası
- **Hürufilik** — hərflərə xüsusi məna verən məzhəb
- **Azərbaycan dilində qəzəl yazımı** (klassik divan azərbaycan dilində)
- **Cəsarətli sufi mövzu** — açıq şəkildə "mən Allaham" deyir
- **Real eşq + ilahi eşq** birləşməsi
- **Şəhid sufi** — sənət üçün ölüm

## Mövzular
- **Vəhdət-vücud** — varlığın birliyi
- **İlahi eşq** (eşqi-haqq)
- **Allahla qovuşmaq** (fənafillah)
- **Hürufilik** — hərf simvolizmi
- **Cəsarətli sufi açıqlama**

## Yazı tərzi
- **Mistik dərinlik** — hər söz çoxqatlı məna
- **İstiarə ustası** — sufi simvolizmi mərkəzdə
- **Cəsarətli ifadə** — "ənəlhəq" tipli açıq sufi mövzu
- **Klassik divan dili** + Azərbaycan ləhcəsi

## İmza misraları (klassik)
```
Məndə sığar iki cahan, mən bu cahana sığmazam,
Gövhəri-laməkan mənəm, kövnü məkana sığmazam.
```
(Mənəm Allahın təcəllisi, hər iki cahan məndə sığar — vəhdət-vücud)

```
Aşiqi-sadiq mənəm, məşuqi-sadiq mən özüm,
Talibi-haqqəm mən özüm, matlubi-haqqəm mən özüm.
```
(Aşiq, məşuq, axtaran və axtarılan — hamısı mənəm)
