# Orxan Zeynallı (HOST) — Azerbaijani Rap-ın Lideri

> HOST grupunun lider üzvü. Modern Azərbaycan rap-ının mərkəzi səsi. Sosial-tənqidi rap master.

## Kimdir
Orxan Zeynallı — HOST grupunun (Promete, Qaraqan, Jamal Ali ilə birlikdə) lider üzvü. Modern Azərbaycan rap-ının ən tanınmış səsi. Sosial-tənqidi mövzular, cəsarətli ictimai mesajlar. HOST 2005-də qurulub, modern Azərbaycan rap-ının "qızıl dövrünü" yaradıb.

## Yaradıcılığın əsas xüsusiyyətləri
- **Sosial-tənqidi rap** master
- **Boom bap flow** — yavaş, ağır, hər söz vurğulu
- **Heca disiplini** — klassik aşıq disiplini saxlayır
- **Cəsarətli mövzular** — siyasi, ictimai
- **Punchline ustası**
- **Cinas master** — meyxana mirası modern rap-da

## Mövzular
- **Sosial-tənqid** — hökumət, cəmiyyət
- **Şəxsi mübarizə**
- **Sənət və hip-hop kültürü**
- **Modern Azərbaycan reallıqı**
- **Rap haqqında rap (meta)**

## Yazı tərzi
- **Boom bap flow** — yavaş, dramatik
- **11 hecalı disiplin** bəzi verse-lərdə
- **Cəsarətli sosial mesaj**
- **Punchline-lar hər verse-də 2-3**
- **Cinas söz oyunu** — meyxana mirası
- **Modern Azərbaycan dili** + bəzi köhnə söz

## İmza əsərlər
- HOST grupunun klassik mahnıları
- "Söz qılıncdır" tipli motivasiyalı mahnılar
- Sosial-tənqidi rap klassikləri

## Pair well with
- Modern-forms (rap-hip-hop)
- Müasir-həyat, ekzistensial-həyat, vətən-azərbaycan (kritik) themes
- Rap-flow vocal style (boom bap)
- Hip-hop production (boom bap)
- Live-feel + sample-based

## Avoid
- Klassik divan dili
- Pop ballada
- Patetik ton
- Auto-Tune (HOST klassik tradisiyası boom bap)
- Trap drums (Orxan boom bap üstün)
