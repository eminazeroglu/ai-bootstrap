# Bəxtiyar Vahabzadə (1925-2009) — Fəlsəfi Şair, Vətəndaşlıq Səsi

> Modern Azərbaycan ədəbiyyatının fəlsəfi səsi. Sovet dövründə cəsarətli vətəndaşlıq şeirləri. "Gülüstan" poeması — Cənub-Şimal Azərbaycan ayrılığı haqqında.

## Kimdir
Bəxtiyar Vahabzadə — Modern Azərbaycan ədəbiyyatının ən hörmətli şairlərindən. **Fəlsəfi şeir master**. Sovet dövründə Azərbaycan vətən mövzusunu cəsarətlə yazırdı — Cənubi Azərbaycan (İran), Qarabağ, milli kimlik mövzular. Akademik şair, alim, ictimai xadim.

## Yaradıcılığın əsas xüsusiyyətləri
- **Fəlsəfi şair** — kompleks fikir
- **Vətən mövzusu cəsarətli** — Cənubi Azərbaycan, Qarabağ
- **Klassik vəzn + modern dil** qarışıq
- **Heca + sərbəst vəzn** dəyişimli
- **Sovet senzurasına qarşı** ədəbi mübarizə
- **Akademik dərinlik**

## Mövzular
- **Vətən** — Cənubi Azərbaycan ayrılığı, Qarabağ, milli kimlik
- **Fəlsəfi-ekzistensial** — həyatın mənası, vaxt
- **Lirik eşq** (kompleks, fəlsəfi)
- **Dil** — Azərbaycan dilinə həsr olunmuş şeirlər
- **Ictimai problemlər** (cəsarətlə)

## Yazı tərzi
- **Möhtəşəm Azərbaycan dili**
- **Klassik vəzn disiplini** + sərbəst vəzn dəyişikliyi
- **Fəlsəfi dərinlik**
- **Lirik incəlik** + ictimai məsələ qarışıq
- **Hörmətli, intellektual ton**

## İmza əsərlər
- **"Gülüstan"** poeması — 1813-cü il Gülüstan müqaviləsi (Cənub-Şimal Azərbaycan bölgüsü) haqqında, Sovet dövrü cəsarətli mətn
- **"Riyaziyyatçı Lobaçevskiyə"** — fəlsəfi
- **"Azərbaycanım"** — vətən mövzusu
- Çoxsaylı qəzəllər və modern lirik şeirlər

## Klassik misal "Gülüstan" mövzusunda
```
Bir vətən, iki yarısı,
Bir dil, iki tərəfi,
Tarixin yarımış izləri —
Bir gün biz birləşəcəyik biz vahid yenidən.
```

## "Mənim dilim" şeiri (Azərbaycan dilinə həsr)
```
Mənim dilim — mənim canım,
Mənim dilim — mənim qanım,
Sənsiz mən kim, sənsiz mən nə —
Sənsən mənim hər nəfəsim, dil canım.
```

## Modern istifadə
- "Gülüstan" mövzusu modern Azərbaycan musiqisində dirilmə
- Fəlsəfi şeirlər intellektual aranjmanda ifa olunur
- Vətən mövzulu mahnılar üçün referans
- Modern lirik şair Vahabzadə davamçılığı (Aqşin Yenisey, Salam Sarvan)

## Pair well with
- Modern-forms (heca + sərbəst qarışıq)
- Vətən-azərbaycan, ekzistensial-həyat themes

## Avoid
