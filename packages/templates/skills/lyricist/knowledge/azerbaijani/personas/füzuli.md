# Məhəmməd Füzuli (1483-1556) — Klassik Divan Ədəbiyyatının Tacı

> Klassik Azərbaycan divan ədəbiyyatının ən böyük şairi. "Leyli və Məcnun" müəllifi. Eşq və hicran ustası.

## Kimdir
Məhəmməd Süleyman oğlu Füzuli — XVI əsr klassik Azərbaycan divan şairi. Üç dildə yazıb: Azərbaycan türkcəsi, fars, ərəb. Üç divan və epik məsnəvi "Leyli və Məcnun" yazıb. Klassik Azərbaycan divan ədəbiyyatının zirvəsidir.

## Yaradıcılığın əsas xüsusiyyətləri
- **Üç dildə yaradıcılıq** (Azərbaycan, fars, ərəb)
- **Eşq mövzusu master** — real + ilahi qarışıq
- **Hicran (ayrılıq) ustası** — klassik divan ədəbiyyatının ən dramatik mövzusu
- **Qəzəl forması zirvə** — Füzuli qəzəlləri klassikadır
- **"Leyli və Məcnun"** — epik eşq əsəri
- **Klassik divan dili master**

## Mövzular
- **Eşq** — real (dünyəvi) və ilahi (sufi) qarışıq
- **Hicran** — ayrılıq əzabı
- **Vüsal** — qovuşma arzusu
- **Sufi mövzular** (bəzi qəzəllərdə)
- **Klassik fəlsəfə**

## Yazı tərzi
- **Möhtəşəm divan dili** — fars-ərəb söz ehtiyatı zəngin
- **Təşbeh və metafora ustası**
- **İstiarə zəngin** — sufi simvolizmi
- **Dramatic dərinlik** — hər beytdə kompleks emosiya
- **Klassik bəhrlərdə mükəmməl** vəzn disiplini

## İmza misraları (klassik)
```
Mən cəfanın tutmuşam dadın, vəfa nə şeydir, bilməm,
Eşqdə təqdir nə imiş bunu sən söylə, mən təfsir bilməm.
```

```
Hicran odunda yandı könül-canım, dad ondan,
Sənsiz bu cahanda mən nə deyim, ağladım.
```

## "Leyli və Məcnun" — epik eşq
Füzulinin ən məşhur əsəri. Real eşq + ilahi eşq qarışıq. Məcnun divanəliyi sufi divanəliyə çevrilir əsər boyunca.

## Modern istifadə
- **Üzeyir Hacıbəyov "Leyli və Məcnun" operası** (1908) — Şərq dünyasının ilk operası, Füzuli mətni əsasında
- Klassik muğam ifaçılarının repertuarında dominant
- Alim Qasımov bəzi qəzəlləri ifa edir
- Müslüm Maqomayev klassik repertuarda Füzuli

## Bu personanı nə zaman seçmək?
- Klassik divan eşq mövzusu
- Hicran (ayrılıq) ən dramatik formada
- Sufi-real eşq qarışıq mövzu
- "Möhtəşəm" klassik atmosfer

## Pair well with
- Classical-forms (qəzəl, müxəmməs, məsnəvi)
- Eşq-məhəbbət, ayrılıq-həsrət, mistik-irfani themes
- Mugham-style vocal
- Tar + kamança + ud + ney
- Orchestral cinematic + Şərq fusion

## Avoid
- Modern pop song structure
- Modern slang
- Lo-fi production
- Trap drums
- Sadə dil (Füzuli möhtəşəm dil tələb edir)
