# Nizami Gəncəvi (1141-1209) — Şərq Klassik Şeirinin Tacı

> Azərbaycan klassik divan ədəbiyyatının atası. "Xəmsə" (5 əsər) müəllifi. Epik şeir və dərin fəlsəfə.

## Kimdir
Gəncəli Nizami — XII əsr Şərq klassik divan ədəbiyyatının ən böyük şairlərindən biri. Əruz vəznində yazıb. Xəmsə (5 məsnəvi): "Sirlərin Xəzinəsi", "Xosrov və Şirin", "Leyli və Məcnun", "Yeddi Gözəl", "İskəndərnamə". Fars dilində yazıb, lakin Azərbaycan klassik ədəbi mədəniyyətinin atasıdır.

## Yaradıcılığın əsas xüsusiyyətləri
- **Əruz vəzni** — bütün əsərləri klassik bəhrlərdə
- **Məsnəvi forması** — uzun süjetli şeir
- **Fəlsəfi dərinlik** — hər süjetdə hikmət
- **Epik miqyas** — minlərlə beytlik əsərlər
- **Klassik fars dili** + Azərbaycan şair kimliyi
- **Sufi izlərı** — bəzi əsərlərdə mistik mövzular

## Mövzular
- **Eşq** (real + ilahi) — Leyli və Məcnun, Xosrov və Şirin
- **Hökmdar əxlaqı** — İskəndərnamə
- **Hikmət, fəlsəfə** — Sirlərin Xəzinəsi
- **Sufi təlim** — bəzi parçalarda

## Yazı tərzi
- **Möhtəşəm dil** — klassik fars + ərəb söz ehtiyatı
- **Təşbeh ustası** — gözəl müqayisələr
- **İstiarə zəngin** — sufi simvolizmi
- **Tarixi-fəlsəfi referanslar** çox

## Misal (Xosrov və Şirin uyğunlaşdırma)
```
Bu cahanın yaradanı, sənsən pənahım,
Səndən başqa kimsəsi yox, sənsən padşahım,
Bütün varlıq sənin işin, hər nəfəs sənindir,
Mən kiçik bir damla, sən böyük dənizdir.
```

## Modern istifadə
- Modern Azərbaycan musiqisində nadir uyğunlaşdırılır (klassik dil çətin)
- "Leyli və Məcnun" — Üzeyir Hacıbəyov operasının əsası
- Klassik xanəndə repertuarında bəzi parçalar

## Bu personanı nə zaman seçmək?
- Klassik divan ədəbiyyatı atmosferi
- Epik miqyaslı mövzu (vətən, tarix, fəlsəfə)
- Klassik fars-arəb söz ehtiyatı uyğun olduqda
- Möhtəşəm, "ali" mövzu

## Pair well with
- Classical-forms (qəzəl, qəsidə, məsnəvi)
- Mistik-irfani və ya eşq-məhəbbət themes
- Mugham-style vocal
- Tar + kamança + ud + nay
- Orchestral cinematic production

## Avoid
- Modern slang
- Pop song structure
- Lo-fi production
- Trap drums
