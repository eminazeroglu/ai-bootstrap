# Əruz Vəzni — Ümumi Baxış

> Klassik divan ədəbiyyatının vəzni. Heca yox, hecalın uzunluq-qısalığına əsaslanır. Nizami, Füzuli, Nəsimi vəzni.

## Tərif
Əruz vəzni — ərəb mənşəli vəzn sistemidir. Hecaların **uzun** (-) və **qısa** (˘) olmasına əsasən qurulur. Nizami Gəncəvi, İmadəddin Nəsimi, Məhəmməd Füzuli — Azərbaycan klassik divan şairlərinin əsas vəznidir.

## Əsas prinsip
- Heca **sayına** baxılmır (heca vəznindən fərqli)
- Hecaların **uzunluq-qısalığına** baxılır
- Açıq heca (sait+samit ilə bitmir) = qısa (˘)
- Qapalı heca (samit ilə bitir) və ya uzun saitli = uzun (-)

## Misal: heca uzunluqları
**Aşiq oldum sənə** (sözləri əruz təhlili):
- a-şiq → ˘ - (qısa, uzun)
- ol-dum → - - (uzun, uzun)
- sə-nə → ˘ ˘ (qısa, qısa)

## Klassik nümunələr

### Füzuli (Bəhri-rəməl: -˘-- / -˘-- / -˘-- / -˘--)
```
Mən cəfanın tutmuşam dadın, vəfa nə şeydir, bilməm,
Eşqdə təqdir nə imiş bunu sən söylə, mən təfsir bilməm.
```
(Hər misrada -˘-- bütün ritm sxemi sabit)

### Nəsimi (Bəhri-həzəc)
```
Məndə Məcnundan füzun aşiqlik istedadı var,
Aşiqi-sadiq mənəm, Məcnun mənim düşmənim.
```

### Nizami Gəncəvi (Bəhri-mütəqarib — "Xosrov və Şirin")
```
Cahanın yaradanı, sənsən pənahım,
Səndən başqa kimsəsi yox, sənsən padşahım.
```

## Əruzun əsas formaları

### 1. Qəzəl
- 5-15 beyt
- Aa, ba, ca, da... qafiyə sxemi
- Mövzu: əsasən eşq, mistik
- Son beytdə təxəllüs

### 2. Qəsidə
- 15-100 beyt (uzun)
- Mövzu: tərif, mədhiyyə, ictimai
- Quruluşu: nəsib + mədhi + duə

### 3. Rübai
- 4 misra (aaba)
- Bəhri-həzəc və ya rübai bəhrində
- Fəlsəfi, hikmət

### 4. Müxəmməs
- 5 misralı bəndlər
- Klassik forma

## Əruz öyrənmə çətinliyi
Əruz öyrənmək heca vəznindən qat-qat çətindir:
1. Hər söz üçün heca uzunluqlarını bilmək lazım
2. Ərəb-fars sözlərinin uzun saitlərini tanımaq
3. 19 bəhrin sxemlərini əzbər bilmək
4. İmalə (qısa hecanı uzun oxumaq) qaydaları
5. Vəsl (iki hecanı bir oxumaq) qaydaları

## Müasir tətbiq

### Əruz az istifadə olunur
XX əsrdə heca vəzni və sərbəst şeirin üstünlüyü ilə əruz arxa plana keçdi. Lakin:
- Klassik mahnılarda (Üzeyir Hacıbəyov "Leyli və Məcnun" operası)
- Müasir muğam ifaçılarının repertuarında (Səid Rüstəmov, Cabbar Qaryağdıoğlu)
- Türk və Fars dünyasında klassik musiqidə hələ də canlı

### Müasir mahnıda əruz izləri
- Aqşin Yenisey bəzi şeirlərində klassik əruz formalarını dirildir
- Şəhriyar əsərlərinin musiqi versiyalarında əruz vəzni dominant qalır
- Mərhum şair Bəxtiyar Vahabzadə bəzi qəzəllərində əruz işlədib

## Əruz haqqında bilməli olunmaq lazım gələn söz növləri

### Türk-Azərbaycan sözləri (qısa heca dominant)
- Çox vaxt qısa hecalar yaradır
- Əruzda uyğunlaşma çətindir

### Ərəb-fars sözləri (uzun heca dominant)
- "Aşiq", "məşuq", "vəfa", "cəfa", "qəm", "ləzzət"
- Əruz üçün təbii uyğun

### Klassik şair seçimi
Klassik əruz şairləri çox vaxt ərəb-fars mənşəli sözlərə üstünlük verirdilər (vəzn məcburiyyəti).

## Qaçılmalı yanlış
- Heca sayına baxmaq (əruzda heca sayı dəyişə bilər)
- Türk-Azərbaycan dili üçün təbii olmayan sözlərə zorla əl atma
- Bəhrlərdən birini oturtmadan yazmaq
- İmalə və vəsl qaydalarını bilmədən cəhd
- Müasir Azərbaycan dili ilə əruz qarışdırma (oxşamır əgər ustad deyilsə)

## Tövsiyə (müasir mahnı üçün)
Müasir Azərbaycan musiqisi üçün əruz **əsasən** köhnəlmişdir. Əgər klassik atmosfer (Füzuli stili, mistik qəzəl) yaratmaq istəyirsən, əruz seç. Əks halda heca vəzni və ya sərbəst şeir uyğundur.
