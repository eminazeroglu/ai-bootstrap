# 8 Hecalı Vəzn — Gəraylı

> Aşıq şeirinin əsas formalarından. 8 heca, 4+4 duraq. Lirik və axıcı.

## Tərif
Gəraylı — 8 hecalı, dörd misralıq bənd quruluşunda olan aşıq şeiridir. Hər bənddə aaab qafiyə sxemi (1, 2, 3 misra qafiyəli, 4-cü misra rədif/refren), və ya abab.

## Quruluş
- Hər misra **8 heca**
- Duraq sistemi: **4+4**
- Bənd ölçüsü: **4 misra** (dördlük)
- Adətən 5-7 bənd olur (orta uzunluq)
- Sonuncu bənddə şair adını çəkir (təxəllüs)

## Qafiyə sxemi

### Klassik gəraylı: aaab + cccb + dddb...
```
a — Səhər oldu, bahar gəldi
a — Çiçəklər bağda görsəndi
a — Bülbül oxudu yenə də
b — Yarımı gözüm görmədi  ← rədif (təkrarlanan misra son)

c — Düzlər yaşıllıq geyindi
c — Buludlar göydən endi
c — Hər tərəf nur, ay sevgilim
b — Yarımı gözüm görmədi  ← rədif təkrar
```

### Müasir gəraylı: abab abab...
Pop musiqisində daha geniş yayılıb.

## Duraq sistemi: 4+4
```
Sevgilim sən, / vətən sənsən  (4+4)
Hər nəfəsim / sənə qonsun  (4+4)
Mən gözündən / can verirəm  (4+4)
Sən mənimsən, / aldım candan  (4+4)
```

## Gəraylının xüsusiyyətləri
- **Lirik mövzular**: əsasən eşq, gözəllik, təbiət, dostluq
- **Axıcı söz**: oxumağa rahat, söz atılmır
- **Aşıq tradisiyası**: Aşıq Ələsgər, Aşıq Şəmşir gəraylı ustaları
- **Müasir uyğunluq**: pop balladalarda və müasir folk-da geniş istifadə

## Aşıq Ələsgər tipik gəraylı formatı

### Tövsiyə (mövzu giriş)
İlk bənddə mövzu açılır, oxucunu çəkir.

### İnkişaf (4-5 bənd)
Mövzu dərinləşir, müxtəlif baxış bucaqları.

### Yekun (son bənddə təxəllüs)
"Ələsgər deyər ki..." formasında şair adını çəkir.

```
Ələsgəri sənə qurban,
Bilirsən bu dünya yalan,
Bir də yarın gəlmir bu yana,
Yarımı gözüm görmədi.
```

## 8 Hecalı Gəraylının Müasir Tətbiqi

### Pop ballada chorus
```
Sənsən mənim / can canımda  (4+4)
Hər ürəyim / döyünəndə  (4+4)
Adın yanır / könlüm odlu  (4+4)
Sənsiz necə / yaşayım mən  (4+4)
```

### Folk-pop verse (Sevda Yahyayeva sayağı)
```
Bahar gəldi, / vətən gözəl  (4+4)
Hər tərəfdə / qaynaqır el  (4+4)
Toylar başlar, / gənclər oynar  (4+4)
Azərbaycan / sülh içində  (4+4)
```

## Rədif (təkrarlanan son söz/cümlə)
Klassik gəraylıda hər bəndin son misrası eynidir — rədif. Bu məsuliyyət:
1. Yaddaşı möhkəmləndirir
2. Mövzunu vurğulayır
3. Mahnıda chorus rolunu oynayır təbii şəkildə
4. Müasir pop "hook" konsepsiyasının kökü

## Müasir musiqidə gəraylı izləri
- **Brilliant Dadaşova** — pop balladalarda 8 hecalı vəzn
- **Manaf Ağayev** — folk-pop tərzində gəraylı
- **Aygün Kazımova** chorus-larında 8 hecalı simmetriya
- **HOST grupu** — bəzi rap verse-lərdə gəraylı disiplini

## Gəraylı yazma metodu
1. Mövzu sec — lirik, mehriban olmalı
2. Birinci bənddə mövzu aç
3. Hər bənddə aaab qafiyə (b — rədif)
4. 4+4 duraqı pozma
5. 5-6 bənd yaz, son bənddə təxəllüs (modern: imza)
6. Söz axarı təbii olsun, zorla qafiyə uyduran kimi olmasın

## Yanlışlar — qaçınmaq
- 7 və ya 9 heca işlətmə
- Duraq pozulması (3+5 və ya 5+3 işlətmə qadağandır)
- Rədifi pozmaq
- Çox uzun bənd yaratmaq (5+ misra qadağan)
- Mövzudan kənar dramatik dönüşlər (gəraylı sakit qalır)

## Klassik nümunə (Aşıq Ələsgər)
```
Ələsgərəm gözüm yaşlı,
Daim qəlbim qəm yarıymış.
Hər kim sevdi, qürbət düşdü,
Yarımı gözüm görmədi.
```
(Sxem: aaab, 4+4 duraq, 8 heca hər misrada)

## İnstrumentlərlə uyğunluq
Gəraylı çox vaxt **saz** müşayiəti ilə oxunur. Pop versiyada **akustik gitara**, **piano** keçərlidir. Tar və kamança ilə də əla işləyir.
