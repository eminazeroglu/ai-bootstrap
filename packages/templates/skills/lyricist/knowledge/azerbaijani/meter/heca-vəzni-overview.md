# Heca Vəzni — Ümumi Baxış

> Azərbaycan xalq şeirinin əsas vəzni. Hecaların sayına və duraqlara əsaslanır. Sadə, axıcı, oxunaqlı.

## Tərif
Heca vəzni — şeir misralarındakı hecaların sayına görə qurulmuş vəzn sistemidir. Hər misrada heca sayı sabitdir. Misra daxilində duraqlar (pauzalar) var.

## Əsas xüsusiyyətlər
- **Heca sayı**: hər misra eyni sayda heca daşıyır
- **Duraq**: misra daxilində bölgüləri yaradan pauzalar
- **Qafiyə**: misra sonlarında səs uyğunluğu
- **Bənd quruluşu**: 2, 3, 4, 5, 6 misralı bəndlər ola bilər
- **Akustik prinsip**: oxumağa, deməyə, yadda saxlamağa rahat

## Heca növləri (sayına görə)
| Heca sayı | Tipik forma | İstifadə |
|-----------|-------------|----------|
| 7 hecalı | bayatı, mani | Qısa, kəskin emosiya |
| 8 hecalı | gəraylı | Aşıq şeiri, lirik |
| 11 hecalı | qoşma, müxəmməs | Klassik aşıq, ən geniş |
| 14 hecalı | divani | Dərin, fəlsəfi |
| 15 hecalı | qəsidə (heca) | Uzun, möhtəşəm |
| 16 hecalı | müxəmməs | Genişlənmiş forma |

## Duraq sistemi (misra bölgüsü)
Duraq misrada nəfəs alma yeridir, ritmin əsasıdır:

**7 hecalı**: 4+3 və ya 3+4
- Misal: "Mən aşıq / o yarıma" (3+4)

**8 hecalı**: 4+4
- Misal: "Sevgilim, vətən / sənsən mənim" (4+4) — sözün düzgün durağı eşidiləndə

**11 hecalı (qoşma)**: 6+5 və ya 4+4+3
- Misal: "Bir gözəl gördüm / al-yanaqlı" (6+5)

**14 hecalı (divani)**: 7+7 və ya 4+4+3+3
- Misal: "Aşiqəm gül üzünə / könlüm həsrətdə qalıb" (7+7)

## Qafiyə sxemləri (heca vəznində)
**aaba** (bayatı, müstəzad bəzən):
```
a — Mən aşıq tər çiçəyə
a — Bənzərim ney çiçəyə
b — İllərdir gözüm yolda
a — Bir əlim ver, gül çiçəyə
```

**abab** (qoşma, modern pop):
```
a — Sevdiyim sənsən, ey gözəl yar
b — Hər gecə düşürsən röyamıza
a — Adın qəlbimdə açan bir bahar
b — Sevgi yazıldı bu süluqamıza
```

**aaab** (gəraylı):
```
a — Səhər oldu, oyandım mən
a — Pəncərədən baxdım gülümsəyən
a — Hər tərəfi nur, ay çiçəyim
b — Səni xatırladıb göndərdim sən
```

## Heca vəzninin gücü
- Söz axarı təbii — danışıq dilinə yaxın
- Yadda qalan — uşaqdan ahala kimi
- Aşıq tradisiyasının əsası
- Müasir rapın da sayqa-əsaslı bölgüsü heca vəzninə uyğun
- Pop musiqisində geniş istifadə

## Müasir tətbiq (Suno üçün)
Müasir Azərbaycan pop / rap mahnılarında çox vaxt:
- 7-8 hecalı qısa misralar (chorus üçün)
- 11 hecalı verse (qoşma sayağı)
- Duraq pozulmadan ritm dəqiq qalır

## Heca pozulması — qaçılmalı yanlış
- Bir misrada heca sayı dəyişməsi (məs: 11+12+11+10) heca vəzninə xəyanətdir
- Duraqlar sıçramamalı (3+4 ilə 4+3 qarışmamalı bənddə ardıcıl)
- Söz axarı zorla — heca üçün söz uyduran şair gəzilir

## Misallar müasir mahnılardan
- HOST (Orxan Zeynallı): rap-da heca disiplini saxlayır
- Müslüm Maqomayev klassik repertuar: 11-hecalı dominant
- Aygün Kazımova pop: 8 hecalı verse + 7 hecalı chorus tipik
