# 7 Hecalı Vəzn — Bayatı və Mani

> Ən qısa, ən kəskin Azərbaycan xalq forması. 4 misra, 7 heca. Bir nəfəsdə deyilən həqiqət.

## Tərif
7 hecalı vəzn — hər misrada 7 heca olan vəzn növüdür. Bayatı və mani bu vəzndə yazılır. Xalq şeirinin ən qədim, ən sıxılmış forması.

## Quruluş

### Bayatı (Azərbaycan ən geniş forması)
- 4 misra
- Hər misra 7 heca
- Qafiyə sxemi: **aaba**
- 1, 2 və 4-cü misralar qafiyəlidir
- 3-cü misra sərbəst (qafiyəsiz)

### Mani (Türk və ya Azərbaycan variantı)
- 4 misra
- 7 heca
- Sxem: aaba (eyni bayatı kimi)
- Mövzu çox vaxt sevgi, ayrılıq, gözəllik

## Duraq sistemi
**4+3** (ən geniş istifadə):
```
Mən aşiqəm / o yara
Çəkməz başım / könülə
Bir xəbər ver / onlara
Köç eyləsin / könülə
```

**3+4** (alternativ):
```
Yaz olur, / qara qaynar
Yarpaq pərqəl / boyatar
Çox ağır / həsrət çəkər
Sevdiyimə / qovuşar
```

## Bayatı növləri

### 1. Sevgi bayatıları
```
Aşıqəm gül üzünə
Bənzəyim ney sözünə
Bir xəbər ver yarımdan
Olum qurban özünə
```

### 2. Ayrılıq bayatıları
```
Mən aşiqəm bahara
Yarpaq düşür kənara
Yar yadıma düşəndə
Su tökərəm odlara
```

### 3. Vətən bayatıları
```
Mənim ana torpağım
Daşı, qumu can canım
Sənsiz nə həyatım var
Sənsiz mənim yox günüm
```

### 4. Hikmət bayatıları
```
Aşiq olan inləməz
Sirrini söz eyləməz
Yar gözündən qan damar
Bir-bir gilə eyləməz
```

## Qafiyə incəlikləri
- **Tam qafiyə**: yara / oxlara (sait + samit eyni)
- **Yarımqafiyə**: dağa / bağa (vurğu yox sait dəyişik)
- **Cinas qafiyə**: yar (sevgili) / yar (ayrıl) — eyni səsli, fərqli mənalı

## 7 Hecalı Vəznin Gücü
- **Yaddaşa rahat**: bir nəfəsdə yadda qalır
- **Emosiyaya kəskin**: az sözlə dərin hiss
- **Şifahi mədəniyyət**: nağılçı, aşıq, ana laylası tradisiyası
- **Müasir rap**: HOST, Miri Yusif bayatı ritmindən istifadə edir
- **Pop chorus**: 7 hecalı söz axarı yaddaqalan hook yaradır

## Müasir tətbiq

### Pop mahnı chorus-u
```
Səni sevdim ay gözəl  (7)
Adın canda titrəyər  (7)
Sən əksiksən hər zaman  (7)
Mənim üçün, ay gözəl  (7)
```

### Rap verse-də bayatı uyğunlaşdırması
```
Söz açıram bir-bir mən  (7)
Söz dəlirəm hər yerdən  (7)
Səhnədəyəm gecə-gün  (7)
Səs ucaldıram yer-yerdən  (7)
```

## Yaranma tarixi
- Çox qədim, şifahi xalq mədəniyyətindən
- Yazılı şəkildə XV-XVI əsrlərdən
- Aşıqlar tərəfindən saxlanıb gəlmə
- Toy, yas, beşik mahnılarında dominant
- Layla, oxşama bayatı şəklində

## Bayatı yazmaq üçün məsləhət
1. Mövzunu birinci misrada qoy
2. İkinci misrada genişləndir
3. Üçüncü misrada hekayə dön (qafiyəsiz)
4. Dördüncüdə güclü zərbə (qafiyə qaytar)

## Müasir mahnılarda bayatı izləri
- Ramiz Rövşən bayatı tradisiyasını müasir şeirə gətirir
- Müsəlbaba ulduzu, Aşıq Şəmşir kimi aşıqlar bayatı master
- Pop balladalarda 7 hecalı chorus dominant qalmaqdadır

## Qaçılmalı yanlış
- 8 və ya 6 heca işlətmə (bayatı pozulur)
- 3-cü misraya qafiyə qoymaq (bayatı qaydası deyil)
- Mənaca uzun cümlə zorlamaq (qısa, kəskin olmalı)
- Modern sərbəst düşüncə qatmaq (bayatı ənənəvi qalmalı)
