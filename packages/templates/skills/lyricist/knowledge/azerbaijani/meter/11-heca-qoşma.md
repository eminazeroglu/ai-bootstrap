# 11 Hecalı Vəzn — Qoşma

> Aşıq şeirinin kralı. 11 heca, 6+5 və ya 4+4+3 duraq. Ən geniş istifadə.

## Tərif
Qoşma — 11 hecalı, dörd misralıq bənd quruluşunda olan klassik Azərbaycan aşıq formasıdır. Hər bənd dörd misra, hər misra 11 heca. Aşıq Ələsgər, Aşıq Şəmşir, Molla Cüməcənin əsas formaları arasındadır.

## Quruluş
- Hər misra **11 heca**
- Duraq sistemi: **6+5** (klassik) və ya **4+4+3** (alternativ)
- Bənd ölçüsü: **4 misra**
- Adi qoşma uzunluğu: 5-7 bənd
- Sonuncu bənddə təxəllüs (şair adı)

## Qafiyə sxemləri

### Klassik qoşma: abab cccb dddb...
1-ci bənd: a-b-a-b
Sonrakı bəndlər: c-c-c-b (b — rədif)

```
a — Aşiqəm gül üzünə canım yarım
b — Bənzəyim ney sözünə, qara gözüm
a — Bir xəbər ver mənə tez ay yarım
b — Yarımı gözüm görmədi  ← rədif

c — Düzlər yaşıl geyindi, dağ aydındır
c — Hər tərəf nur içində, gül zərlidir
c — Bülbül oxudu yenə də gül üstündə
b — Yarımı gözüm görmədi  ← rədif təkrar
```

### Müasir qoşma: abab cdcd...
Hər bənd müstəqil qafiyə, daha azad.

## Duraq sistemi

### 6+5 (ən geniş)
```
Aşiqəm gül üzünə / canım yarım  (6+5)
Bənzəyim ney sözünə / qara gözüm  (6+5)
Bir xəbər ver mənə tez / ay yarım  (6+5)
Yarımı gözüm görmədi  (5+6 alt — fərqli durar bəzən)
```

### 4+4+3 (alternativ)
```
Aşiqəm gül / üzünə canım / yarım  (4+4+3)
Sevirəm gül / yanağa, gözüm / dolur  (4+4+3)
```

## Qoşmanın xüsusiyyətləri
- **Geniş mövzu spektri**: eşq, fəlsəfə, vətən, hicran, şikayət, pəh
- **Aşıq sənətinin bel sütunu**: ən çox bu vəzndə əsər var
- **Dramatik qurulma**: mövzu açılır, dərinləşir, kulminasiyaya çatır, yekunlaşır
- **Təxəllüs**: ustad aşıqlar son bənddə adlarını çəkir

## Aşıq Ələsgərdən klassik qoşma nümunəsi

```
Bahar gəldi, bağlar yenə cənnətdir,
Bülbül oxur gül üstündə, lalada,
Hər bir tərəf çiçəkli zinətdir,
Yarımı gözüm görmədi.

Ələsgərəm sözüm tamam dedimsə,
Bu dünyaya bel bağlayıb qalmaqdı,
Sevda bilmiş, könül atəş ilə yandı,
Yarımı gözüm görmədi.
```

## Müasir tətbiq: Pop ballada / folk-pop

### Verse (qoşma sayağı)
```
Bu şəhərdə tanıyırsan / hər küçəni  (6+5)
Hər kafedə içdik birgə / şirin çayı  (6+5)
İndi tək gəzirəm o / yerləri yenə  (6+5)
Sənsiz hər şey yad olub mən / üçün, yarım  (6+5)
```

### Chorus (qısa-formaya keçir)
Verse 11 hecalı qoşma, chorus isə 7-8 hecalı bayatı/gəraylı keçidi tipikdir.

## 11 Hecalı Qoşmanın Gücü
- **Bəstəkar üçün**: melodiyaya geniş yer verir (bir misraya çox not düşə bilər)
- **Söz üçün**: kompleks fikir bir misrada çata bilər
- **Hekayə anlatımı**: bir bənddə mini-səhnə ola bilər
- **Aşıq sənəti**: improvizasiya üçün ideal struktur

## Müasir musiqidə qoşma izləri
- **Aşıq Ələsgər** — bütün repertuar qoşma əsaslı
- **Aşıq Pənah** — müasir aşıq qoşma davamçısı
- **Brilliant Dadaşova** — bəzi mahnılarda qoşma vəznli verse
- **Süleyman Ələsgərov** bəstəladırdığı mahnılar — qoşma poetik baza
- **HOST grupu rap-i** — bəzi verse-lərdə qoşma disiplini

## Qoşma yazma metodu
1. **Mövzu seç** — qoşma geniş, dramatic mövzu daşıya bilər
2. **Birinci bənddə açılış** — oxucunu çək
3. **Orta bəndlərdə inkişaf** — 3-4 bənd
4. **Son bənd** — kulminasiya + təxəllüs (klassik) və ya imza (müasir)
5. **Duraq pozma** — 6+5 sistemi sabit qal
6. **Rədif gözəl seç** — yaddaqalan bir cümlə (klassik versiya üçün)

## Yanlışlar — qaçınmaq
- 10 və ya 12 heca işlətmə
- Duraq sıçraması (5+6, 7+4 və s. işlətmə)
- Rədifi pozmaq və ya hər bənddə dəyişmək (klassik versiyada)
- Mövzudan dəfələrlə dönüş (qoşma fokuslu qalmalı)
- Çox sadə dil (qoşma nəcib, zərif söz tələb edir)

## Müxəmməs ilə fərq
- **Qoşma**: 4 misralı bənd, 11 heca
- **Müxəmməs**: 5 misralı bənd, 11 və ya 16 heca

## İnstrumentlərlə uyğunluq
- **Saz** (klassik aşıq müşayiəti)
- **Tar + kamança** (klassik mahnı versiyası)
- **Akustik gitara + piano** (müasir uyğunlaşdırma)
- **Folk-pop band** (bütöv ansambl)
