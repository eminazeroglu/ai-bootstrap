# Tərcibənd — Təkrarlanan Keçid Beytli Klassik Forma

> Tərkibbəndin qardaşı. Hər bənddə eyni "tərci beyti" təkrarlanır. Mövzu vurğulaması güclü.

## Tərif
Tərcibənd — klassik divan ədəbiyyatının mürəkkəb şeir formasıdır. Tərkibbəndlə oxşardır, lakin əsas fərqi: **hər bəndin sonunda təkrarlanan EYNİ tərci beyti** (keçid beyti) gəlir. Bu, mərkəzi fikrin daimi vurğulanmasını yaradır.

## Quruluş
- **Hər bənd**: 5-10 misralıq qəzəl-tipi quruluş
- **Hər bəndin sonunda**: 1 tərci beyti (2 misra)
- **Tərci beyti hər bənddə EYNİ qalır** (tərkibbənddən əsas fərq)
- **Vəzn**: əruz (klassik bəhrlər)
- **Bənd sayı**: 5-10 bənd

## Sxem
```
Bənd 1:
  - 5-10 misra (qəzəl quruluşunda)
  - Tərci beyti (2 misra) — A və B

Bənd 2:
  - 5-10 misra (yeni qafiyə qrupu)
  - TƏRCİ BEYTİ — eyni A və B (təkrar)

Bənd 3:
  - 5-10 misra
  - TƏRCİ BEYTİ — eyni A və B (təkrar)
...
```

## Klassik nümunə (uyğunlaşdırma)

### Tərci beyti (bütün bəndlərdə təkrarlanan)
```
Eşq sənətdir, hər ürək onu öyrənə bilməz,
Yalnız sevən bilər ki, eşq nədir, necə yenilməz.
```

### Bənd 1
```
Eşq qoyub məni dünyaya, gözəl gözünə qurban,
Hər nəfəsim sənindir, sevda mənim canıma can,
Köhnə dərdləri unutdum, sən gəldin mənə təzə,
Ay üzünə baxan göz, görmür ay yanağa zəhmət,
Bir təbəssüm versən, qurtular bütün bu hicran,
Gözlərində dolu sevda, sənsən mənim ürəyim, can.

Tərci beyti:
Eşq sənətdir, hər ürək onu öyrənə bilməz,
Yalnız sevən bilər ki, eşq nədir, necə yenilməz.
```

### Bənd 2
```
Hicran odu yandırdı könlümü, dağıtdı sirrlərimi,
Hər gecə yuxusuz qalmışam, gözüm gözləyir səni,
Səssiz küçələrdə dolaşıram, axtarıram izini,
Sənin yoxluğun bir kölgə, daim dolaşır mənimlə,
Yarı qaldı bütün xəyallarım, bütün arzularımız,
Bir tərəfdə mən, bir tərəfdə sən — qarışmaz suyumuz.

Tərci beyti (təkrar — eyni):
Eşq sənətdir, hər ürək onu öyrənə bilməz,
Yalnız sevən bilər ki, eşq nədir, necə yenilməz.
```

## Tərcibəndin gücü
- **Mərkəzi fikrin daimi vurğulanması** — tərci beyti hər bənddə eyni qalır
- **Refrain effekti** — modern mahnıdakı "chorus" konsepsiyasının kökü
- **Yaddaşa rahat** — təkrarlanan beyt sayəsində formaya yaxınlıq
- **Mövzu birlik** — bütün bəndlər eyni mərkəzi fikrə bağlıdır

## Tərcibənd vs Tərkibbənd (kritik fərq)
| | Tərcibənd | Tərkibbənd |
|---|---|---|
| Keçid beyti | **Hər bənddə EYNİ** (təkrarlanır) | **Hər bənddə FƏRQLİ** |
| Effekt | Refrain, vurğu | Çoxsəsli inkişaf |
| Çətinlik | Bir az asan (təkrar var) | Daha çətin (yeni keçid hər dəfə) |
| Mövzu | Mərkəzi bir fikir | Çoxqatlı mövzu inkişafı |

## Klassik tərcibənd nümunələri
- **Saib Təbrizi** sufi tərcibəndləri
- **Fuzuli** divanından bəzi tərcibəndlər
- **Vidadi**, **Ziya Paşa** (Türk klassik) — tərcibənd ustaları

## Mövzular
- **Mistik / sufi** (ilahi eşq, Allah-vəhdət)
- **Lirik** (eşq, hicran)
- **Fəlsəfi** (vaxt, ölüm, dünyanın faniliyi)
- **İctimai mərasimdə** (mərhum-yaslı tərci-bəndlər)

## Müasir tətbiq

### Mahnıda chorus konsepsiyası
Modern mahnıdakı **chorus** məhz tərci beytinin uyğunlaşdırılmış formasıdır:
- Verse-lər mövzunu inkişaf etdirir
- Chorus (tərci beyti) hər dəfə eyni şəkildə təkrarlanır
- Mərkəzi fikir vurğulanır

### Pop ballada strukturu
```
[Verse 1] — qəzəl-tipi quruluş, hekayə açılır
[Chorus] — TƏRCİ BEYTİ, mərkəzi fikir
[Verse 2] — yeni hekayə, eyni mövzu
[Chorus] — eyni tərci beyti
[Bridge] — fərqli açı
[Final Chorus] — tərci beyti son dəfə
```

### Rap-da
HOST grupu və başqa Azərbaycan rapçıları "hook" konsepsiyasında tərci beyti ilkin tərkibini istifadə edirlər.

## Tərcibənd yazma metodu
1. **Mərkəzi fikir tap** — bütün şeirin xülasəsi olacaq tərci beyti üçün
2. **Tərci beyti yaz** — 2 misra, möhtəşəm və mənalı
3. **Vəzn seç** — bütün bəndlər eyni bəhrdə qalmalı
4. **Hər bənd qəzəl kimi qur** — öz qafiyə qrupu, mövzunu fərqli açıdan yansıtsın
5. **Tərci beyti hər bənddən sonra eyni təkrarla** — dəyişdirmə
6. **5-7 bənd yaz** — mövzu çox açılan ola bilər
7. **Son bənd zirvə** — tərci beyti yekun olaraq daha güclü təsir bağışlamalıdır

## Yanlışlar — qaçınmaq
- Tərci beytini bənddən-bəndə dəyişdirmək (bu tərkibbənd olur)
- Bəndlər arası mövzu birlik qurmamaq
- Vəzni dəyişdirmək
- Tərci beytini sadə cümlə kimi yazmaq (möhtəşəm olmalıdır)
- Heca vəznində yazıb "tərcibənd" adlandırmaq

## Tərcibəndin müasir əhəmiyyəti
Müasir mahnıda **chorus konsepsiyası** birbaşa tərcibənd formasından gələn miras hesab oluna bilər. Hər dəfə eyni şəkildə təkrarlanan mərkəzi fikir mahnını yaddaqalan edir.

## İnstrumentlərlə uyğunluq
- **Tar + kamança + ud** (klassik müşayiət)
- **Bütöv muğam ansamblı**
- **Orkestr** (möhtəşəm aranjman)
- **Modern band ensemble** (verse-chorus uyğunlaşdırması üçün)
- **Piano ballad arrangement** (kamera versiyası)
