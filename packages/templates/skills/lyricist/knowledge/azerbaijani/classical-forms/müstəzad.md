# Müstəzad — Əlavə Misralı Klassik Forma

> Hər əsas misradan sonra qısa "əlavə misra" gəlir. Klassik divan ədəbiyyatının nadir, xüsusi forması.

## Tərif
Müstəzad — klassik divan ədəbiyyatının orijinal formasıdır. Hər **əsas misranın** sonuna **qısa əlavə misra** (yarımmisra və ya kiçik misra) əlavə olunur. "Müstəzad" sözü ərəbcə "artırılmış" mənasını verir.

## Quruluş
- **Əsas misra** + **əlavə misra** cütləşir
- Əsas misralar əruz vəznində (uzun bəhri)
- Əlavə misralar həmin bəhrin yarısı (qısa hissəsi)
- Hər cüt birgə bir vahid təşkil edir
- Qafiyə sxemi: əsasən qəzəl tipidir, əlavə misralar da öz qafiyə qaydasında

## Vizual quruluş
```
[ÜZÜN ƏSAS MİSRA buradadır, geniş fikir daşıyır]
    [qısa əlavə misra — vurğu, şərh, və ya əlavə]
[ÜZÜN ƏSAS MİSRA, davam edir]
    [qısa əlavə misra]
```

## Klassik nümunə (uyğunlaşdırma)
```
Eşq qoyub məni dünyada gözəl gözünə qurban olum,
    Hər nəfəsim sənindir.
Köhnə dərdləri unutdum, sən gəldin, çiçək kimi açıldım,
    Sevda cana cəlb olur.
Bir təbəssüm versən mənə, bütün hicran daşıram, daşı,
    Gözlərində nur saçır.
Yarımı görmək istəyirəm hər səhər oyananda mən,
    Lakin yox, hələ uzaq.
```

## Müstəzadın xüsusiyyətləri
- **İki səviyyəli mətn** — əsas + əlavə birgə oxunur
- **Vurğu effekti** — əlavə misra əsasa şərh kimi durur
- **Visual ritm** — uzun-qısa-uzun-qısa səslənir oxunduqda
- **Klassik nadir** — ən az yazılan klassik formalardan biri
- **Sənətkarlıq sınağı** — uğur təbii axar tələb edir

## Mövzular
- **Lirik** (eşq, hicran)
- **Mistik** (sufi məcaz)
- **Fəlsəfi** (dünyanın fanilik müşahidələri)
- **Şərhli mətn** (əlavə misralar şərhi yarada bilər)

## Klassik Azərbaycan müstəzadı
Çox az yazılıb. Bəzi divan şairlərinin (Saib Təbrizi, Vidadi) divanlarında izləri var. Forma çox çətin olduğundan ümumi populyarlığı az olub.

## Müasir tətbiq

### Mahnıda call-and-response
Müstəzadın "əsas misra + əlavə misra" konsepsiyası modern call-and-response strukturuna uyğun gəlir:
- Solo vokal əsas misranı oxuyur
- Backup vokal və ya xor əlavə misranı oxuyur
- Yaxud verse + tag formatı

### Pop-rok hibrid struktur
```
[Verse]
Bu küçələrdə tanıyıram hər izi sənin...
    (kim bilir hara getdin?)
Hər kafedə içdik birgə şirin çayı bir vaxt...
    (indi mən təkəm orada)
```

### Spoken-word / rap kombinasiyası
- Əsas misra rap formasında
- Əlavə misra spoken-word kimi (ad-libs, vocal asides)
- HOST grupu və modern Azərbaycan rapçıları belə struktur istifadə edirlər

## Müstəzadın diğer formalarla əlaqəsi
- **Qəzəl + əlavə**: əslində "genişləndirilmiş qəzəl"dir
- **Qoşma + nidalar**: heca vəznində oxşar effekt yaratmaq olar
- **Rap call-and-response**: modern uyğunlaşdırma

## Müstəzad yazma metodu
1. **Vəzn seç** — uzun bəhri (rəməl, həzəc) əsas misralar üçün
2. **Əsas misraları yaz** — qəzəl tipi
3. **Əlavə misraları əsas misraların yarı uzunluğunda yaz**
4. **Əlavə misralar əsasa "şərh" və ya "vurğu" qatsın**
5. **Qafiyə sxemini saxla** — əsas + əlavə öz aralarında ahənglə
6. **Oxuyub yoxla** — əsas + əlavə təbii axsa, uğurludur

## Yanlışlar — qaçınmaq
- Əsas + əlavə misraları eyni uzunluqda yazmaq (müstəzad pozulur)
- Əlavə misraları sadəcə əsasdan kəsmək (məna olmamalı qarmaqarışıq)
- Vəzn pozma — əruz disiplini saxlanmalı
- Heca vəznində yazıb "müstəzad" adlandırmaq

## Müstəzadın gücü
- **Çoxsəsli forma** — əsas + əlavə iki səs yaradır
- **Vizual maraq** — sətirlər fərqli uzunluqda göz üçün diqqətcəkici
- **Modern dramatik** — bugünki spoken-word + rap üçün tarixi əsas
- **Sənət iddiası** — müstəzad yazan şair texniki ustalıq nümayiş etdirir

## Müstəzad nümunəsi (tam)
```
Bahar gəldi, çiçəklər açıldı yenə hər tərəfdə,
    Yarım hardadır mən bilmirəm.
Bülbül oxudu gül üstündə nəğmələr şirin və xoş,
    Mənim ürəyim dolu kədər.
Hər nəfəsim sənindir, gözəl gözlərinin qurbanıyam,
    Bilirsən, mənə güc ver.
Bir gün gələrsən, bilirəm, sənsiz yaşayanmam mən,
    Sənsiz hər şey bir yarımdır.
```

## İnstrumentlərlə uyğunluq
- **Tar + kamança** (klassik müşayiət)
- **Soloist + backup vokal** (call-and-response struktur)
- **Müasir orkestr** (dramatic forma uyğunlaşdırması)
- **Spoken-word + minimal piano** (modern intim versiyası)
- **Rap + electronic backing** (post-modern uyğunlaşdırma)

## Müstəzad bugün niyə vacibdir?
- **Klassik miras** — Azərbaycan ədəbi tarixinin orijinal formalarından
- **Modern uyğunluq** — call-and-response və spoken-word strukturunun klassik kökü
- **Sənət iddiası** — müstəzad yazmaq texniki bacarıq nümayişidir
- **Innovasiya** — müasir Azərbaycan mahnısında müstəzad uyğunlaşdırması orijinal estetik yarada bilər
