# Qəsidə — Klassik Mədhiyyə Şeiri

> Uzun (15-100 beyt), möhtəşəm, çox vaxt patron tərifi və ya ictimai mövzulu klassik forma. Qəzəlin "böyük qardaşı".

## Tərif
Qəsidə — ərəb mənşəli, uzun və möhtəşəm klassik şeir formasıdır. Qəzəllə eyni qafiyə sxemində (aa, ba, ca, da...) yazılır, lakin daha uzun olur (15-100 beyt) və mövzu daha geniş olur. **Mədhiyyə** (təriflə), **fəxriyyə** (şair özünü tərif), və ya **müşahidə** (cəmiyyət təsviri) növlərinə bölünür.

## Quruluş
- **15-100 beyt** (təxminən 30-200 misra)
- Hər beyt 2 misra, əruz vəznində
- Qafiyə sxemi: **aa, ba, ca, da, ea...** (qəzəl ilə eyni)
- Üç hissəyə bölünür:
  1. **Nəsib** (giriş — təbiət təsviri, atmosfer)
  2. **Mədhi** (əsas hissə — tərif və ya mövzu)
  3. **Duə** (yekun — şəxsiyyətə xeyir-dua)

## Hissələr detallı

### 1. Nəsib (giriş — 5-15 beyt)
- Təbiət təsviri (bahar, payız, gecə, səhər)
- Atmosfer yaratmaq
- Oxucunu çəkmək
- Mövzuya yumşaq keçid

### 2. Mədhi (əsas — 15-70 beyt)
- Tərif edilən şəxsiyyət (sultan, vəzir, hökmdar)
- Bütün məziyyətləri sayma
- Mübaliğəli təriflər
- Tarixi müqayisələr (İsgəndər kimi, Süleyman kimi)

### 3. Duə (yekun — 5-15 beyt)
- Tərif edilənə xeyir-dua
- Uzun ömür, sağlamlıq, qələbə
- Şair təxəllüsü çəkilir

## Mövzu növləri

### 1. Mədhiyyə qəsidəsi
Patron tərifli (ən geniş yayılmış)

### 2. Fəxriyyə qəsidəsi
Şair özünü tərif edir, sənətini mədh edir

### 3. Hicr qəsidəsi (kədərli)
Vəfat etmiş şəxsə həsr (ölü mərasimi şeiri)

### 4. Tövbə qəsidəsi
Şairin tövbə etməsi, dini mövzu

### 5. Müşahidə qəsidəsi
Cəmiyyət, əxlaq, hadisələr təsviri

## Klassik Azərbaycan qəsidə ustaları
- **Xaqani Şirvani** (XII əsr) — qəsidənin tacı, "Qəsidə-i Şini" (250+ beyt)
- **Nizami Gəncəvi** — bəzi qəsidələr xəmsənin müqəddiməsində
- **Füzuli** — bəzi qəsidələr
- **Saib Təbrizi** (XVII əsr) — sufi qəsidələri
- **Müsəddiq** klassiki — XIX əsr Azərbaycan qəsidə davamçısı

## Misal: Xaqani Şirvani tərzi (uyğunlaşdırma)
```
Nəsib (giriş):
Bahar gəldi, çiçəklər açıldı, tabiət yenildi,
Bülbül oxudu nəğmələr, gül üstündə savaş başlandı...

Mədhi (əsas):
Hökmdarımız, ey ədalət sahibi, sənsən günəş qalqan,
Sənsiz ərzimiz qaranlıq, sənsiz gəlir hər zaman bahar yenidən...

Duə (yekun):
Səndə uzun ömür, ey hökmdar, qızıl ulduz kimi parlaq,
Xaqani sənə dua edir, qələmiylə yazır bu qadağanı.
```

## Müasir tətbiq

### Klassik dirilmə (nadir)
Müasir Azərbaycan ədəbiyyatında qəsidə demək olar ki yazılmır. Lakin:
- Bəzi müasir şairlər vətənə həsr olunmuş "qəsidə-tipli" əsərlər yazır
- Səməd Vurğun "Azərbaycan" şeiri qəsidə ruhuna yaxındır

### Mahnıda qəsidə izləri
- Vətən mövzulu uzun mahnılar (Üzeyir Hacıbəyov "Sənsən, ey Vətənim")
- Hökmdara həsr olunmuş klassik xanəndə repertuarı

## Qəsidə vs Qəzəl
| | Qəsidə | Qəzəl |
|---|---|---|
| Beyt sayı | 15-100 | 5-15 |
| Mövzu | Mədhiyyə, ictimai, böyük | Lirik (əsasən eşq) |
| Quruluş | Nəsib + Mədhi + Duə | Atmosfer bütöv |
| Ton | Möhtəşəm, ali | İntim, dərin |
| Xitab | Şəxsiyyətə (sultan, hökmdar) | Sevgili (real və ya mistik) |

## Qəsidə yazma metodu (müasir tələbə üçün)
1. **Mövzu seç** — mədh edilməyə layiq şəxsiyyət və ya konsept (vətən, ana, sənət)
2. **Bəhr seç** — uzun forma üçün rəməl və ya kamil bəhri uyğun
3. **Nəsib yaz** — 5-7 beyt atmosfer yaradan giriş
4. **Mədhi yaz** — 10-20 beyt əsas mövzu
5. **Duə yaz** — 3-5 beyt yekun + təxəllüs
6. **Vəzn pozma** — bütün qəsidə eyni bəhrdə qalmalı
7. **Qafiyəni təkrar et** — bütün ikinci misralar eyni qafiyə + rədif

## Yanlışlar — qaçınmaq
- 15-dən az beyt (qəsidə uzun olmalıdır)
- Mövzu dəyişikliyi (qəsidə bir mövzuya fokuslaşır)
- Sadə dil (qəsidə klassik, möhtəşəm dil tələb edir)
- Strukturu pozmaq (nəsib + mədhi + duə üçlüyü vacibdir)
- Heca vəznində yazmaq (qəsidə əruz tələb edir)

## Qəsidənin gücü
- **Möhtəşəmlik** — qrandiyozluq və geniş forma
- **Tarixi sənəd** — patron tərifi tarixi qaynaqdır
- **Söz ehtiyatı** — çoxlu söz, çoxlu təsvir
- **Sənət ölçü** — qəsidə yaza bilən şair "ustad" sayılırdı klassik dövrdə

## İnstrumentlərlə uyğunluq
- **Tar + kamança + nağara** (klassik möhtəşəm müşayiət)
- **Orkestr** (Üzeyir tərzi opera-cüstü)
- **Xor + soloist** (anlayış-cüstü struktur)
- **Müasir film orkestrası** (Hans Zimmer tərzi qəsidə uyğunlaşdırması)
