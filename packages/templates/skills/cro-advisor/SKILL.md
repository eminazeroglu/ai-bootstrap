## GTM motions

| Motion | ACV | Cycle | CAC |
|---|---|---|---|
| **Self-serve** | <$2K | days | $50-500 |

| **Hybrid** | mixed | mixed | mixed |

Pick ONE primary motion before scaling.

## Sales funnel stages

```
Lead → MQL → SQL → Opportunity → Customer
       ↑     ↑     ↑              ↑
   marketing-qualified  sales-qualified   closed-won
```

Define each stage explicitly. Mismatch = pipeline lies.

## Quota setting

- Bottom-up: rep capacity × win rate × ACV
- Top-down: revenue target ÷ rep count
- Reconcile: take the LOWER (don't over-commit)

Quota attainment 60-80% = healthy.
Above 100% = quotas too low.
Below 50% = bad hires OR bad quotas OR bad product.

## Compensation design

- Base 50%, variable 50% (early)
- Base 40%, variable 60% (scale)
- Accelerators above 100%
- SPIFs for specific products
- Clawback if customer churns <3 months

## Pipeline math

```
Pipeline coverage = (Pipeline $ / Quota) 
3× = healthy
4-5× = comfortable
<3× = pipeline crisis
```
