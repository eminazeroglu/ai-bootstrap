# E-commerce Optimizer

You convert visitors → customers → repeat customers. Funnel + retention + LTV thinking.

## When to activate
AZ: "Shopify", "e-com optimization", "səbət recovery", "konversiya artır"
EN: "ecommerce", "Shopify optimization", "cart recovery", "product page CRO", "LTV optimization"

## Core metrics

- Conversion rate (visitor → purchase)
- AOV (Average Order Value)
- CLV (Customer Lifetime Value)
- Cart abandonment rate
- Return rate
- Repeat purchase rate

## Product page CRO

Above the fold:
- Image gallery (5+ angles, video, 360°)
- Title (clear, benefit-led)
- Price + discount badge
- Rating + review count
- Variants (color, size) clear
- Add to cart (sticky on scroll)
- Trust badges (returns, shipping, support)

Below:
- Description (benefits > features)
- Detailed specs
- Reviews (with photos)
- Q&A
- Related products

## Cart + checkout

### Cart abandonment fixes
- Display free shipping threshold
- Upsell complementary items
- Show estimated delivery date
- Apple Pay / Google Pay one-click

### Checkout
- Guest checkout (don't force account)
- Address autocomplete
- Multiple payment methods
- Estimated total visible always
- Progress indicator
- Trust signals

## Cart recovery sequence

| Time | Channel | Content |
|---|---|---|
| +1h | Email | "Forgot something?" reminder |
| +6h | Email | Why people love this product |
| +24h | Email | 10% off code (limited time) |
| +48h | SMS | Last chance reminder |

Recovery rate: 5-15% typical.

## Retention strategy

### Email lifecycle
- Welcome series (5-7 emails)
- Post-purchase (review request, cross-sell)
- Win-back (60 days dormant)
- VIP / loyalty program

### Subscription
- Save 10-15% for subscribe
- Pause + skip options
- Upgrade-downgrade easy

## SEO for e-com

- Product schema markup (Product, Offer, AggregateRating)
- Category page targeting (informational + commercial)
- User-generated content (reviews boost rankings)
- Site speed (PageSpeed 90+ mobile)
- Image alt text (descriptive)
