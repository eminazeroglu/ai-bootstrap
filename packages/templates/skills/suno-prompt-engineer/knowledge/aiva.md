# AIVA — Model Specification

> AI Virtual Artist. Specialized in symphonic / orchestral / film score composition. MIDI output capable.

## Model identity
- **Provider**: AIVA Technologies (aiva.ai)
- **Generates**: Composed musical scores (instrumental focus)
- **Output format**: MP3 / WAV / MIDI / MusicXML
- **Cost basis**: Subscription (Free / Standard / Pro)
- **Founded**: 2016 (one of the earliest AI music tools)
- **Specialty**: Classical, orchestral, cinematic

## Strengths
- **MIDI output** (huge for composers — editable in DAW)
- **MusicXML output** (sheet music editable)
- Strong on classical/orchestral idiom
- Strong on film score / underscore
- Style presets (cinematic, modern classical, jazz, etc.)
- Mood-based generation (specify emotion)
- Length control (15s to 5+ min)
- Commercial license clarity (paid tiers)
- "Influence" feature (upload MIDI to influence style)

## Weaknesses
- Vocals: not supported (instrumental only)
- Modern pop / hip-hop / electronic: weaker
- Less "AI-feel" innovative — more "composer assistant"
- Output sometimes generic-orchestral
- Multilingual not relevant (instrumental)

## Workflow

### Generation modes
1. **Preset-based**: pick style (cinematic, modern classical, etc.)
2. **Influence-based**: upload reference MIDI/audio
3. **From scratch**: describe mood + tempo + key + duration

### Example prompt parameters
- **Style**: Cinematic / Modern Classical / Pop / Jazz / Tango / Folk / etc.
- **Mood**: Epic / Sad / Joyful / Mysterious / Romantic / Tense
- **Duration**: 15s to 5+ min
- **Key signature**: pickable
- **Time signature**: pickable
- **Tempo**: BPM specify

## Best uses
- **Film score** drafts / placeholder
- **Game music** (background, ambient, themes)
- **YouTube background** (royalty-free use)
- **Composer drafting** (then refine in DAW via MIDI)
- **Educational** (study composition AI-generated)
- **Commercial** (paid tier covers usage rights)

## MIDI advantage (most important)
The MIDI export means YOU can:
- Open in Logic / Ableton / FL Studio / Cubase
- Replace AI-generated instruments with your samples
- Edit notes individually
- Combine with your own composition
- Use as starting point, not final product

This makes AIVA more of a **composer assistant** than a final-output tool.

## Output editing
- Web editor available (basic)
- DAW import via MIDI (full power)
- Sheet music export for live performers
- Stem export (paid tier)

## Best practices
- Use as DRAFT generator, then refine
- Combine multiple AIVA generations
- Use influence mode with your reference (more personal output)
- Short generations + manual stitching beats long single-gen
- Treat MIDI output as composition skeleton

## When to use AIVA vs Suno/Udio/Stable Audio
- **AIVA**: orchestral / film score, MIDI editability, classical idiom
- **Suno**: full songs with vocals
- **Udio**: hip-hop / electronic instrumental
- **Stable Audio**: ambient, loops, sound design

## License tiers
- **Free**: limited downloads, attribution required
- **Standard**: more downloads, no attribution
- **Pro**: full commercial rights, MIDI export

## Anti-patterns
- Don't use for vocal songs
- Don't use for modern pop production
- Don't expect cutting-edge experimental output
- Don't ignore MIDI export — it's the killer feature
- Don't use as final product without DAW refinement (typically)
