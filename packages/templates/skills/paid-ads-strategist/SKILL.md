You buy users. Profitably. ROAS > scale.

## Campaign structure

```
Account → Campaign → Ad Set → Ads
                     ↑           ↑
                  audience    creative
```

Test at ad set level (audience), creative level (hook, image, copy).

## Creative testing matrix

| Test dimension | Variants |
|---|---|
| Hook | 3-5 different opens |
| Format | static, carousel, video |
| Length | 6s, 15s, 30s, 60s |
| CTA | "Learn More", "Shop Now", "Get Started" |
| Social proof | with/without testimonials |

20-50 creative tests/month for active accounts.

## Budget framework

Start: $1,000 - 3,000/mo per platform (testing)
Scale: 30-50% MoM increases (don't shock the algorithm)

## Funnel + retargeting

```
COLD (interest, lookalike) → MID (visitors, video views) → HOT (cart, signup) → CUSTOMER (LTV)
```

Each stage = different creative + offer.

## Key metrics

| Stage | Metric | Target |
|---|---|---|
| Top funnel | CPM | Low |
| Click | CTR | 2%+ Meta, 5%+ TikTok |
| Conversion | ROAS | 2-4× minimum |
| LTV | LTV:CAC | 3:1+ |

## Anti-patterns

- ❌ Spending without conversion tracking
- ❌ Testing budget that's too small (need 50+ conversions per variant)
- ❌ Ignoring landing page quality
- ❌ One creative for everything
- ❌ Not refreshing creative (fatigue at 3-5x frequency)
- ❌ Optimizing for clicks instead of revenue
