# Icon & Graphic Elements — When/Why/How (Sərt qaydalar)

> Mövzu icon-ları və graphic element-lər **opsional**dır — host face + text həmişə əsas elementdir. Icon **yalnız o vaxt** əlavə olunur ki, real **topic recognition** və ya **curiosity gap completion** verir. Tə default şəkildə əlavə etmək amateur signal-dır.

## Fundamental qayda

### Icon əlavə etmə qərar matrisi

| Şərt | Qərar |
|---|---|
| Mövzu universal-tanınan visual signal-a sahibdir (fitness = dumbbell, money = $, time = clock) **VƏ** text + face hələ də mövzu signal etmir | ✅ İcon ƏLAVƏ ET |
| Text + face onsuz da mövzu signal edir (məs. host fitness zalında dumbbell tutmuş) | ❌ İcon ƏLAVƏ ETMƏ (redundant) |
| Mövzu abstract (philosophy, opinion, commentary) — universal icon yoxdur | ❌ İcon ƏLAVƏ ETMƏ |
| Composition artıq doludur (3+ element) | ❌ İcon ƏLAVƏ ETMƏ (clutter) |
| Niche-də icon norma deyil (vlog, music, beauty) | ❌ İcon ƏLAVƏ ETMƏ |
| Niche-də icon norma-dir (gaming, finance, sports, comedy) | ✅ Yoxla — uygundursa əlavə et |

**Default:** İcon **YOXDUR**. Yalnız konkret səbəb olduqda əlavə olunur.

---

## When icons HELP (✅ əlavə et)

### 1. Instant topic recognition
Text-i oxumadan əvvəl izləyici dərhal mövzunu başa düşür:
- 💰 (money bag) + "100K" → "money / income content"
- 🎮 (controller) + "WIN" → "gaming"
- ⏰ (clock) + "30 SECONDS" → "time-pressure / fast"
- 🍞 (bread, food-themed) + "FIT" → comedy fitness food joke
- 📈 (chart up) + "10X" → "growth / investment"

**Example: Variant A cvn-salamatı-budu — "BU NƏ DƏ 200 MANAT?!"**
- Universal money icon (₼ AZ symbol və ya 💰 money bag) text yanında dərhal "price content" signal edir
- Text-i oxumadan dərhal mövzu (qiymət reaksiyası) tanınır → curiosity gap maksimum
- **Qərar: ✅ ƏLAVƏ ET** — small "₼200" badge sağ-üst tərəfdə (text-dən ayrı, lakin support)

### 2. Curiosity gap completion
İcon sual qoyur, text cavab verir (və ya əksinə):
- Şok üzü + ❓ icon + text → "what happened?" gap aktiv
- Clock + ZAMAN MAŞINI text → time-mystery setup

### 3. Brand consistency
Kanalın recurring icon-ı (məs. MrBeast həmişə dollar icon) → brand recognition signal

### 4. Niche convention (genre signal)
- Gaming → controller / game-specific logo (industry norm)
- Finance → $ / chart / coin (industry norm)
- Sports → ball / medal / team logo (industry norm)
- Comedy → optional burst shapes ("BOOM!" balon), reaction icons

## When icons HURT (❌ əlavə etmə)

### 1. Redundancy
Subject zaten icon-u **physically tutur** və ya **wears**:
- Host dumbbell tutmuşsa → dumbbell icon LAZIM DEYIL
- Host saatı bilek-də → clock icon LAZIM DEYIL
- Host pul nümayiş etdirir → 💰 icon LAZIM DEYIL

### 2. Composition clutter
3+ element artıq var (host face + text + background context):
- Əlavə icon = göz dağılır
- Hierarchy pozulur
- 120px mobile-da hiç-bir element oxunmur

### 3. Generic emoji clip-art look
Bu icons amateur signal-dır:
- 🔥💯😱🤯 (emoji dump)
- Generic free clip-art (Microsoft Office 2000 vibe)
- Cartoon stickers (Snapchat-tier)

**Doğru icon style:**
- Custom outlined vector OR
- High-quality flat illustration OR
- Brand-specific glyph (custom-designed)

### 4. Niche mismatch
- Vlog → icon əlavə etmək authenticity pozur
- Music → icon əlavə etmək performance fokusunu zəiflədir
- Beauty → makeup product icon yerinə real product göstər (foto)
- Education → emoji icon yerinə **visual metaphor** (Veritasium pattern)

### 5. Cultural noise
Cross-cultural ambiguous icon:
- 🙏 (Christianity / Buddhism / "please" — context-dependent)
- 👌 (OK in West, vulgar in some cultures)
- Hand gestures (high cultural sensitivity)

## Icon style guidelines

### Visual hierarchy
- **Size:** 15-25% of frame height (face həmişə daha böyükdür)
- **Position:** rule of thirds intersection və ya text-dən ayrılmış slot
- **Z-order:** behind face, in front of background
- **Padding:** min 5% from text və face

### Color & contrast
- Background dark → icon white + drop shadow
- Background light → icon dark + bright fill
- Brand-related → brand color (Apple silver, Samsung blue)
- Always: 4:1 minimum contrast ratio

### Style coherence
- Outlined OR filled (qarışıq YOX)
- Vector (NEVER raster bitmap icon)
- Consistent stroke weight across multiple icons
- Same visual language (flat / 3D / cartoon — bir-birinə uygun)

## Topic-specific icon library (recommended)

### Comedy / Sketch
- 💥 burst shape (yellow, comic-book) — reaction explosion
- ❓ question mark balloon — curiosity gap
- 💰 / ₼ money symbol — price comedy
- 🍞 / 🍕 food icon — food comedy
- ⏰ clock — time joke
- 🎬 clapboard — meta-comedy (production reference)
- onomatopoeia text-icons: "BOOM!", "POW!", "WHAT?!", "VAY!" in burst shape

### Tech
- Apple logo / Samsung / Google logos (brand)
- 🔋 battery, 📷 camera, 🖥️ screen, 💾 storage
- Spec badges: "5G", "4K", "8GB"
- Comparison arrows (vs)

### Money / Finance
- $ / ₼ / € / ¥ — currency symbols
- 📈 chart up arrow, 📉 chart down
- 🏦 bank, 💼 briefcase, 💎 diamond (luxury)
- Number badges: "1M", "$10K", "100×"

### Time / Speed
- ⏰ clock, ⏱️ stopwatch
- ⏳ hourglass
- 🚀 rocket (fast)
- "30 SEC", "1 HOUR" text badges

### Gaming
- 🎮 controller, 🕹️ joystick
- Game-specific logos (Fortnite, Minecraft, etc.)
- "WIN", "RAGE", "PRO" text badges
- 💀 skull (rage / fail moments)

### Sports / Fitness
- 🏋️ weight, 🏃 runner, ⚽ ball
- 🏆 trophy, 🥇 medal
- ❤️ heart (cardio)
- Number badges: "100kg", "10K"

### Food / Cooking
- 🍳 cooking, 🍞 bread, 🥗 salad
- ⭐ rating, 🔥 spicy
- Chef hat icon

### Travel / Adventure
- ✈️ plane, 🗺️ map, 📍 location pin
- 🏔️ mountain, 🌊 wave

## Anti-patterns to absolutely avoid

1. **Emoji wallpaper** — thumbnail-ı 5+ emoji ilə doldurmaq (Snapchat-tier amateur)
2. **Random sticker dump** — heç bir mövzu əlaqəsi olmayan dekorativ elementlər
3. **Clip-art aesthetic** — Microsoft Office 2000 vibe (low-resolution, dated)
4. **Cultural insensitivity** — context-asılı icon-lar (gesture, religious)
5. **Icon larger than face** — icon DOMINANT olduqda subject identity itir
6. **Generic when topic deserves custom** — Generic 💰 əvəzinə layihə-spesifik "₼" badge daha güclü
7. **Mismatched style** — outlined + filled + 3D qarışıq
