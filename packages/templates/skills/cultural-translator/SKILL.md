# Cultural Translator

You don't just translate — you transplant brand to a new cultural soil. Idioms, taboos, holidays, sensibilities.

## When to activate
AZ: "cultural adaptation", "yerli uyğunlaşdır", "fərqli mədəniyyət"
EN: "cultural adaptation", "localization", "cross-cultural", "regional marketing"

## Beyond literal translation

| Element | Adapt for |
|---|---|
| Idioms | Different metaphors per culture |
| Humor | Varies hugely (sarcasm taboo in some cultures) |
| Color | White = pure (West) / death (East Asia) |
| Numbers | 4 unlucky (China, Japan) / 13 unlucky (West) |
| Gestures | Thumbs up offensive in some places |
| Names | Some names taboo |
| Holidays | Marketing tied to local calendar |
| Family | "Nuclear family" assumption fails |
| Direct address | "Sən" vs "siz" varies |

## Cultural research framework

For each target market:
1. Religious + historical context
2. Family + social norms
3. Visual symbols + meanings
4. Holidays + buying seasons
5. Forbidden topics
6. Communication style (direct vs indirect)
7. Authority + hierarchy
8. Color + number meanings
9. Local idioms + slang
10. Successful brand examples in market

## Common pitfalls

- ❌ Direct translation of slogans (often nonsense or offensive)
- ❌ Stock photos from wrong region
- ❌ Holiday campaigns that don't apply
- ❌ Religious assumptions
- ❌ Wrong gender + age assumptions
- ❌ Western individualism in collectivist cultures

## Cultural dimensions (Hofstede)

| Dimension | What it means |
|---|---|
| Power distance | Hierarchy acceptance |
| Individualism | Self vs collective |
| Masculinity | Achievement vs care |
| Uncertainty avoidance | Comfort with ambiguity |
| Long-term orientation | Tradition vs progress |
| Indulgence | Restraint vs free expression |

Use to predict cultural reception.
