## Content pillar framework

Lock 4 pillars (mix every week):

```
40%  Education     — how-to, tips, behind-the-scenes
30%  Entertainment — stories, humor, trending
20%  Inspiration   — case studies, transformations
10%  Promotion     — direct sale, offer
```

## Reel structure (3-phase, Meta's playbook)

```
0-3s: HOOK         — stop scroll (text + visual)
3-7s: SUBSTANCE   — promised value start
7-15s: PAYOFF     — deliver value + transition
End:   CTA         — single ask (DM, link, save)
```

### Hook formulas
- Text overlay: "POV: you just discovered..."
- Pattern interrupt: unexpected visual
- Question: "Have you ever wondered..."
- Bold claim: "Stop doing X. Do Y instead."
- Story open: "I learned this the hard way..."

## Stories funnel pattern (5-7 frames)

```
Frame 1: HOOK              — stop scroll
Frame 2: STATS / PROOF     — credibility
Frame 3: PROBLEM           — pain point
Frame 4: SOLUTION          — your approach
Frame 5: CASE STUDY        — real example
Frame 6: OFFER             — what + why now
Frame 7: CTA               — link / DM
```

Use Stories interactive stickers (poll, slider, quiz, question) on Frame 1-2 for boost.

## DM funnel automation

1. **Trigger**: keyword in DM ("info", "qiymət")
2. **Message 1**: auto-reply with link/menu
3. **Message 2** (if no response 24h): nudge
4. **Message 3** (if no response 3d): break sequence

Use Meta Manychat / ChatPlace for AZ market.

## Analytics — what to watch

| Metric | Target |
|---|---|
| Reach / followers | >50% weekly |
| Reel watch-through | >50% |
| Story completion rate | >70% |
| Saves per post | 2-5% of reach |
| Shares per post | 0.5-2% of reach |
| Profile visits → follow | >15% |

Saves + shares > likes (engagement quality > quantity).

## Growth levers (priority order)

1. **Reels quality** — watch-through is everything in 2026
2. **Hook discipline** — first 3s decide
3. **Consistency** — algorithm rewards weekly cadence
4. **Niche clarity** — algorithm needs to know who to show
5. **Engagement velocity** — respond to comments in first hour
6. **Collaborations** — co-Reels with adjacent creators
7. **Story funnels** — convert viewers to DMs/website

## Anti-patterns

- ❌ Generic AI-generated content (audience detects)
- ❌ Hashtag spam (30 generic tags)
- ❌ Buying followers (engagement crashes)
- ❌ Posting without hooks (skip rate >70%)
- ❌ DM blast for cold leads (account suspension risk)
- ❌ Long captions without payoff line 1
- ❌ Single-pillar content (algorithm bored)
