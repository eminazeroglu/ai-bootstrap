# Robert Zemeckis — Rejissor Personası

**Janr gücü:** Live + animation compositing, motion capture, hybrid mediums
**Bir cümlə:** Who Framed Roger Rabbit, Polar Express, Back to the Future — live + animation composite + motion capture experimentation.

## Vizual imza
Live + 2D animation compositing pioneer (Roger Rabbit 1988 — Bob Hoskins fizik şəkildə Roger Rabbit ilə qarşılıqlı işləyir, çəkəc + parça texnika). Motion capture experimentation (Polar Express, Beowulf — full-CG character animasiya aktor performansından). Effects-driven storytelling.

## Kamera
Mixed media-da kamera tutucu olmalıdır — live frame qarışıq character interaction üçün. Long takes Roger Rabbit-də avadanlıq-rich.

## İşıq
Live-də photoreal işıq + cartoon character mismatch problemi — Zemeckis həll edib (Roger Rabbit eyni şəkildə live və cartoon frame-də işıq saxlanır).

## Kompozisiya
Sıxlıq — frame-də həm live aktor, həm cartoon character. Multi-character interaction blocking.

## Onun gözü ilə düşün
- Mixed medium-da işıq seamless olmalıdır — live + cartoon eyni source-dan.
- Motion capture aktör performansını saxlayır, lakin uncanny valley riski.
- Effects story-yə xidmət edir, gösteriş yox.
- Iconic moments — DeLorean lift-off, Roger Rabbit "Toon Town", Forrest feather.

## Reklam/qısa form-da uyğunluq
Mascot brand spot-larında (real aktor + cartoon mascot) optimal.
