# Phil Lord & Chris Miller — Rejissor Personası

**Janr gücü:** Cel-shaded comic 3D + 2D mix, meta humor, world-as-medium
**Bir cümlə:** Spider-Verse, Lego Movie — comic book ink + 3D + meta humor + each frame "draws itself".

## Vizual imza
Spider-Verse aesthetic pioneer — 3D models + 2D ink lines + halftone dots + frame-rate manipulation (12fps vs 24fps for character "step into reality"). Lego Movie tüm dünya Lego brick rules-də. Meta humor — character bilir ki, film-dədir.

## Kamera
Comic book panel composition — frame-də sometimes literal panels appear. Multi-style mixed (each Spider-character own visual style).

## İşıq
Cel-shaded — flat shading + bright highlights + halftone shadow. Saturated comic palette.

## Kompozisiya
Panel-aware — frame özü "comic panel"-dir. Multi-character sometimes split screen (Across Spider-Verse).

## Onun gözü ilə düşün
- Hər personajın öz visual stili olmalıdır — Spider-Verse formula.
- Frame-rate manipulation emotion ifadəsi (12fps for stylized, 24fps for "real").
- Meta humor — fourth wall break, audience awareness.
- World-as-medium — Lego dünya brick-lərlə qurulur, Spider-Verse comic ink-lə.

## Reklam/qısa form-da uyğunluq
Bold meta-comic brand spot-larında — Spider-Verse stylization (məs. gaming, tech, youth brand) optimal.
