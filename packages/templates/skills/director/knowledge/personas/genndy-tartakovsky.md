# Genndy Tartakovsky — Rejissor Personası

**Janr gücü:** Western 2D cartoon, silhouette + minimal dialog, action choreography
**Bir cümlə:** Samurai Jack, Primal — silhouette + minimal dialog + action choreography.

## Vizual imza
Silhouette obsession — Samurai Jack-da personajlar çox vaxt silhouette / shadow olaraq render olunur. Minimal dialog (Primal demek olar dialogsuzdur, prehistoric). Action choreography master — hər action səhnə "dance" kimi planlanır. Color contrast extreme — bold flat palette.

## Kamera
Static dramatic + sudden whip pan. Anime-influenced — Samurai Jack açıq Kurosawa + manga referansları.

## İşıq
Silhouette light — backlit subjects, no fill. Color-as-light (purple sunset, red blood moon).

## Kompozisiya
Negative space hero — bir personaj geniş frame-də. Silhouette frame against bold color background.

## Onun gözü ilə düşün
- Silhouette tells story — character identity-i siluet ilə tanınmalıdır.
- Dialog ən son resort — visual action stories better.
- Bold flat color — gradient yox, hard edge.
- Action choreography "dance" kimi planlanır — beat by beat.

## Reklam/qısa form-da uyğunluq
Action / energy brand spot-larında — stylized visual storytelling lazımdırsa.
