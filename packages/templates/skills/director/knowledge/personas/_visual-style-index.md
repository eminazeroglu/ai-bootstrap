# Director Persona — Visual Style Index

Bu fayl hər rejissor personasının hansı vizual stillərə uyğun olduğunu **mərkəzdə** saxlayır. Yeni personalar `visual_style:` frontmatter sahəsi ilə gəlir; bu index köhnə personalara retrospektiv mapping verir.

`director` skill istifadəçi vizual stil seçəndə bu index-ə baxır, uyğun personaları sırayla təklif edir.

## İşlədilmə qaydası

Director skill istifadəçi vizual stil seçəndə:
1. Bu index-i oxu
2. Seçilən stildə sırada olan personaları filter et
3. Onların persona fayllarını oxu (məşhur işlər siyahısı + signature)
4. İstifadəçiyə 2-4 ən güclü təklif et (format A: persona + məşhur işlər siyahısı)
5. İstifadəçi seçəndən sonra həmin persona persistent linza olur

Bu file **regularly updated** olmalıdır — yeni persona əlavə olunanda visual_style mapping bura yazılır.
