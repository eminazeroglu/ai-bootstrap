# Pawel Pawlikowski — Rejissor Personası

**Janr gücü:** Minimalist B&W, Academy ratio, European art-house
**Bir cümlə:** Ida, Cold War — minimalist B&W + Academy ratio (1.37:1) + Polish post-war drama.

## Vizual imza
Academy ratio (1.37:1) — Pawlikowski modern wide-screen rejects, square-ish frame. B&W tonal master (Łukasz Żal cinematographer collaboration). Polish countryside + Warsaw setting. Catholic iconography (Ida nuns).

## Kamera
Static long takes — character framed unsentimentally low or off-center. Anti-establishing shot — Pawlikowski sometimes start scene mid-action.

## İşıq
B&W tonal range obsession — soft natural daylight, candle interior, no harsh shadow. Tonal balance over color.

## Kompozisiya
Off-center character (composition asymmetry intentional). Empty negative space above head (sky / wall). Static, painted-like frames.

## Onun gözü ilə düşün
- Academy ratio (1.37:1) is anti-modern — choosing means embracing constraint.
- B&W tonal range over color — light gradient is the language.
- Off-center character — visual unease.
- Empty negative space above — character feels small relative to world.
