# Recraft v3 — Image Model Knowledge

## Provider
Recraft

## Type
Diffusion (multi-style)

## Strengths

- Excellent text/typography rendering
- Vector + raster output
- Brand design specialty
- Multiple style modes (realistic, illustration, vector, 3D)
- Logo and identity work
- Clean professional output

## Weaknesses

- Less photorealistic than Nano Banana for portraits
- More design-oriented than cinematic
- Reference image support limited
- Style mode locks aesthetic

## Best Practices

1. **Choose style mode first** (realistic_image, vector_illustration, digital_illustration)
2. Specify typography exactly when needed
3. Use for brand/marketing visuals
4. Be specific about color palette (hex codes work)
5. Specify dimensions clearly
6. Brand consistency via style mode

## Prompt Anatomy

```
Style mode: [realistic_image | vector_illustration | digital_illustration]
+ [Subject and composition]
+ [Brand/color guidance]
+ [Typography if needed]
+ [Aspect ratio]
```

## Sample Prompts

### Brand poster
```
Style: digital_illustration
Minimalist poster for "Karavan" coffee brand. Stylized illustration
of a single coffee cup, steam rising, against deep navy background.
Brand color palette: navy #0A1F44, gold accent #D4A574, cream
#F5F1E8. Typography: brand name "Karavan" in serif font at top,
tagline "fuel for what you carry" in light sans below. 9:16
vertical poster format.
```

### Product shot
```
Style: realistic_image
Product photography of a vintage glass coffee carafe on dark wood
table, soft window light from left, neutral background, shallow
depth of field, premium commercial style. 1:1 square format.
```

### Vector logo
```
Style: vector_illustration
Logo concept for "Karavan" coffee: stylized camel silhouette merged
with a coffee bean shape, gold (#D4A574) on navy (#0A1F44),
flat design, minimalist, scalable vector style. Square 1:1.
```

## Style Modes Explained

- `realistic_image` — photorealistic
- `digital_illustration` — modern illustration
- `vector_illustration` — flat scalable graphics
- `3d` — 3D rendered look
- Specific sub-styles available within each

## Best Use Cases

- Brand identity work
- Marketing posters
- Social media graphics with text
- Logo concepts
- Product mockups
- UI/UX inspiration
