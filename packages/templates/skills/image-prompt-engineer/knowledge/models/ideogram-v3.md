# Ideogram v3 — Image Model Knowledge

## Provider
Ideogram

## Type
Diffusion

## Strengths

- Best-in-class text rendering (most reliable)
- Logo and brand identity work
- Magic Prompt feature (auto-enhances prompts)
- Multiple style presets
- Aspect ratio flexibility
- Reasonable cost

## Weaknesses

- Photorealism behind Nano Banana / Flux
- Less cinematic look out-of-box
- Limited control over photographic style
- Less effective for portrait work

## Best Practices

1. Use for any prompt with text/typography
2. Specify exact text in quotes
3. Magic Prompt can enhance — but disable for precision
4. Style presets limit but speed up consistency
5. Aspect ratio explicit
6. Brand colors via hex codes

## Prompt Anatomy

```
[Subject + composition]
+ [Text content "in quotes"]
+ [Style preset reference]
+ [Color palette]
+ [Aspect ratio]
```

## Text Rendering Tips

- Put exact text in DOUBLE QUOTES
- Specify font style: serif, sans-serif, script, display
- Specify weight: bold, light, regular
- Color of text explicitly

## Magic Prompt

When ON, Ideogram enhances your prompt with style additions.
When OFF, your prompt is followed literally.

For precision work: Magic OFF
For exploration: Magic ON
