# Imagen 4 — Image Model Knowledge

## Provider
Google

## Type
Diffusion

## Strengths

- Excellent text rendering (signs, captions, typography)
- Photorealistic outputs
- Strong prompt adherence (literal interpretation)
- Clean, professional defaults
- Good with technical/architectural subjects

## Weaknesses

- Less cinematic by default (more "stock photo" feel)
- Conservative outputs (less risk-taking)
- Limited stylization compared to Midjourney
- Reference image support evolving

## Best Practices

1. Be explicit about cinematic intent ("cinematic photograph")
2. Specify all elements clearly (Imagen interprets literally)
3. Use camera + lens + aperture for depth
4. Reference photographers for style ("in the style of Annie Leibovitz")
5. Specify aspect ratio in prompt
6. Lighting setup must be explicit
7. Use for product/architectural shots strength

## Prompt Anatomy

```
[Cinematic intent declaration]
+ [Subject + detailed physical description]
+ [Setting + environment]
+ [Lighting setup + color temp]
+ [Camera + lens + aperture]
+ [Style notes + aspect ratio]
```

## Sample Prompt

```
Cinematic photograph in 9:16 vertical aspect: a 35-year-old
Azerbaijani man, athletic build, short dark wavy hair, green eyes,
wearing a charcoal wool overcoat. Standing by a rain-streaked window
in a 1960s American diner, looking outside with melancholic
expression. Lighting: warm tungsten ceiling pendant at 3200K from
upper-left creating rim on shoulders, cool blue spill from rain-
streaked window at 5600K behind. Shot on Sony A7 IV with Cooke S4
85mm lens at f/1.4. Photorealistic, cinematic color grading with
warm shadows and cool highlights, shallow depth of field, slight
film grain.
```

## Text Rendering Strength

Imagen 4 excels at typography:
- "Vintage neon sign reading 'OPEN' in red cursive script"
- "Newspaper front page with headline 'STORM APPROACHES'"
- "Restaurant menu board with daily specials in chalk handwriting"

Use this for brand work.
