# Synth Pad — Instrument

> Sustained atmospheric synthesizer. Provides texture, warmth, atmosphere. Background element.

## Type
Sustained synthesizer sound (pads). Often slow attack and release.

## Sub-types
- **Warm pad** — analog warmth (vintage)
- **String pad** — synthesized strings
- **Bell pad** — bell-like overtones
- **Choir pad** — synthesized choir
- **Atmospheric pad** — abstract textures
- **Granular pad** — modern granular synthesis

## Sound character
- Sustained (slow attack)
- Atmospheric
- Background (rarely lead)
- Wide stereo image
- Lush

## Function in arrangement
- Fills harmonic space
- Provides emotional bed
- Adds atmosphere
- Supports vocals
- Builds tension (with filter sweeps)

## Playing styles
- Sustained chords (whole notes / longer)
- Slow chord changes
- Filter sweeps (open/close over time)
- Subtle modulation

## Genre fit
- Electronic (essential)
- Ambient (foundational)
- Modern pop (atmosphere)
- Cinematic (mood)
- R&B (smooth bed)
- Modern hip-hop (atmospheric)

## Effects
- Heavy reverb
- Delay
- Chorus (lush)
- Filter sweeps

## For Suno
```
[Synth pad: warm analog pad, sustained chords, atmospheric background, 
heavy reverb, lush atmosphere]
```


## When to use
Atmospheric moments. Behind vocals (subtle). Building tension. Cinematic mood. Almost any electronic context.
