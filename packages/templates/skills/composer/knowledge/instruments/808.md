# 808 (Sub-Bass) — Instrument

> Iconic sub-bass sound from Roland TR-808 drum machine. Defines modern hip-hop and trap.

## Type
Synthesized sub-bass drum sound. Originally from Roland TR-808 (1980).

## Sound character
- Deep sub-frequencies (felt physically)
- Long sustain possible
- Slidable pitch (the "drill slide")
- Boomy
- Synthetic but iconic

## Variations
- **Boomy 808** — long sustain (modern trap)
- **Punchy 808** — short attack (cleaner)
- **Distorted 808** — saturated (modern trap)
- **Sliding 808** — pitch bend (drill signature)

## Playing styles
### Modern trap
- Sustained 808s (held for bars)
- Sub-bass drone
- Pitch slides

### Drill
- Sliding 808s (the drill bass slide)
- Distinctive movement

### Classic hip-hop
- 808 kick on beat 1
- Punchier shorter version

## Genre fit
- Hip-hop (modern)
- Trap (essential)
- Drill (sliding signature)
- Modern R&B
- Electronic / EDM
- Modern pop (subtle)

## Production notes
- Sub-bass needs proper mix (subwoofer audible)
- Layer with higher kick for click
- Ducks under kick (sidechain) sometimes
- Defining element of modern productions

## For Suno
```
[808: boomy sub-bass, sustained notes, modern trap style]
```

```
[808: sliding drill bass, pitch bends down on each note]
```

## When to use
Modern hip-hop / trap / R&B. Essential for that "modern" sound. Avoid in acoustic / classical / folk genres.
