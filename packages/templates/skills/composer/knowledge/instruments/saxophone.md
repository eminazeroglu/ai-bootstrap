# Saxophone — Instrument

> Reed instrument. Smooth, sensual, jazzy. Multiple sizes.

## Type
Single-reed woodwind. Brass body (despite woodwind classification).

## Sub-types
- **Soprano sax** — highest, lyrical (Kenny G, classical)
- **Alto sax** — bright, jazz standard (Charlie Parker)
- **Tenor sax** — most common, warm (John Coltrane)
- **Baritone sax** — low, deep (jazz, ska)

## Range
Each saxophone has ~2.5 octaves. Together they cover wide range.

## Sound character
- Smooth, expressive
- Vocal-like
- Can be sensual or aggressive
- Smoky / sultry quality

## Playing styles
### Jazz
- Improvisation central
- Swing or straight
- Long melodic lines
- Bebop fast lines

### Smooth jazz
- Sensual ballads
- Long sustained notes
- Romantic

### Soul / R&B
- Solo lines
- Punctuating accents

### Rock
- Bruce Springsteen "Born to Run" style
- Punctuating moments

### Funk
- Tight rhythmic punches
- Section play

## Genre fit
- Jazz (essential)
- R&B / Soul
- Funk
- Smooth jazz
- Rock (occasional)
- Film score (smoky moments)

## For Suno
```
[Saxophone: tenor sax solo, smooth jazz style, melodic improvisation, 
warm and sensual]
```

```
[Saxophone: alto sax bebop, fast melodic runs, jazz combo]
```

## When to use
Jazz contexts. Smooth romantic moments. Soul/R&B authenticity. Saxophone solo for emotional peak.
