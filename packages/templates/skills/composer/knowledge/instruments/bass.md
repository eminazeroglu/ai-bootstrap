# Bass — Instrument

> Low-frequency foundation. Rhythm + harmony bridge. Essential for groove.

## Type
Bass guitar (electric) or upright (acoustic) or synth bass.

## Range
~2 octaves (typically).

## Sub-types
- Electric bass (most common modern)
- Upright / double bass (jazz, classical, rockabilly)
- Synth bass (electronic, pop)
- Fretless bass (smooth slides)
- 5-string / 6-string (extended low)

## Playing styles
- Root notes (simple holds)
- Walking bass (jazz)
- Slap and pop (funk)
- Fingerstyle (most pop/rock)
- Pick (punk, harder rock)
- Sub-bass (electronic, modern)

## Genre fit
- Pop, rock (essential)
- Jazz (upright walking bass)
- Funk (slap bass essential)
- R&B / Soul (groovy)
- Electronic (synth bass / 808)
- Hip-hop (808 sub-bass)

## Sound character
- Acoustic upright: warm, woody
- Electric: depends on type/pickups
- Synth bass: variable (sub, distorted, etc.)
- 808: huge sub-bass (felt physically)

## For Suno
```
[Bass: electric bass, fingerstyle groove, pocket]
```

```

```

## When to use
Almost every modern song. Bass = backbone.
