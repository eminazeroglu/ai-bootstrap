# Trumpet — Instrument

> Brass instrument. Bright, powerful, declarative. Jazz and orchestral standard.

## Type
Brass instrument. Lip-buzzed.

## Range
F#3 to D6 (standard B-flat trumpet).

## Sound character
- Bright, brilliant
- Powerful, declarative
- Can be aggressive or warm
- Cuts through mix

## Playing styles
### Classical
- Lyrical melodies
- Fanfares
- Orchestral parts

### Jazz
- Improvised solos
- Bebop runs
- Lyrical ballads
- Muted (Miles Davis style)

### Latin
- Mariachi (often pairs)
- Salsa (section)

### Soul / Funk
- Section punches
- Brass hits

### Hip-hop
- Sample source (often)

## Mutes
- Cup mute (warm, soft)
- Harmon mute (Miles Davis cool)
- Plunger mute (jazz expressive)

## Genre fit
- Classical / orchestral
- Jazz (essential)
- Latin (mariachi, salsa)
- Soul / funk (sections)
- Film score (heroic)
- Cinematic moments

## For Suno
```
[Trumpet: lyrical jazz solo, muted Miles Davis style, smoky atmosphere]
```

```
[Trumpet: heroic fanfare, brass section, orchestral cinematic]
```

## When to use
Heroic / triumphant moments (orchestral). Jazz character. Latin authenticity. Brass section accents.
