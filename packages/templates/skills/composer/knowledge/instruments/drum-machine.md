# Drum Machine — Instrument

> Electronic drum sounds. Programmed beats. Hip-hop/electronic foundation.

## Type
Electronic percussion synthesizer / sampler.

## Famous drum machines
- **Roland TR-808** — iconic boomy kick (hip-hop, R&B)
- **Roland TR-909** — house/techno staple
- **Linn Drum** — 80s pop classic
- **Akai MPC** — sample-based (hip-hop production)
- **Modern: Native Instruments Maschine, Ableton Drum Rack**

## Sound character
- Synthetic (vs acoustic)
- Programmable (any pattern)
- Punchy and consistent
- Sub-bass kick possible (808)

## Playing styles
- Programmed grids (4/4 typical)
- Hip-hop boom-bap (kick + snare)
- 4-on-the-floor (house, EDM)
- Trap rolls (hi-hat triplets)
- Glitch / chopped patterns

## Genre fit
- Hip-hop (essential)
- Electronic (all sub-genres)
- Modern pop
- R&B
- Trap (specifically 808 + hi-hats)

## Sub-elements often specified separately
- 808 (see 808.md)
- Trap hi-hats (see trap-hihat.md)

## For Suno
```
[Drum machine: TR-808 kick + clap snare + hi-hats, hip-hop boom-bap pattern]
```

```
[Drum machine: TR-909 4-on-the-floor, house style]
```

## When to use
Modern productions, electronic genres, hip-hop. Replace acoustic drums for synthetic feel.
