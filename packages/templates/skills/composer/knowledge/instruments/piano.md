# Piano — Instrument

> Most versatile keyboard instrument. Works in nearly every genre.

## Type
Acoustic keyboard. Hammers strike strings.

## Range
~7 octaves (88 keys standard).

## Genre fit
- Classical (foundational)
- Pop (chord accompaniment + melody)
- Jazz (essential)
- R&B / Soul
- Singer-songwriter
- Film score
- Indie / folk

## Sub-types
- Grand piano (concert hall)
- Upright piano (intimate)
- Honky-tonk (vintage saloon)
- Prepared piano (experimental)

## Playing styles
- Block chords
- Arpeggios
- Melody lead
- Bass + chords (jazz left hand)
- Rolling figures (classical)

## Mood pairings
- Romantic, melancholic, contemplative, triumphant, peaceful

## For Suno
```
[Piano: solo grand piano, sustain pedal, gentle arpeggios, intimate]
```

## When to use
Almost any song benefits from piano. Default keyboard choice.
