# Acoustic Guitar — Instrument

> Steel-string or nylon-string acoustic. Singer-songwriter standard. Folk foundation.

## Type
Acoustic chordophone. Plucked or strummed.

## Range
~3-4 octaves.

## Sub-types
- Steel-string (folk, country, pop)
- Nylon string / classical (Spanish, classical)
- Dreadnought (full body, loud)
- Parlor guitar (smaller, intimate)
- 12-string (lush, jangly)

## Playing styles
- Strummed chords (folk, pop)
- Fingerpicked (intimate, melodic)
- Hybrid picking
- Slide (blues, country)
- Travis picking (country)

## Genre fit
- Folk, indie folk
- Country
- Singer-songwriter
- Pop ballads
- Spanish / Latin (nylon string)
- Acoustic rock

## Mood pairings
- Intimate, romantic, melancholic, peaceful, nostalgic

## For Suno
```
[Acoustic guitar: fingerpicked steel-string, warm tone, intimate]
```

```
[Acoustic guitar: strummed dreadnought, folk feel, drive]
```

## When to use
Default instrument for intimate / folk / singer-songwriter contexts.
