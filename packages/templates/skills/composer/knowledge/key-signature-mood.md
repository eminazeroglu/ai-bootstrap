# Key Signature & Mood — Music Knowledge

> Key signature defines emotional palette. Critical compositional choice.

## Major keys (generally bright)

| Key | Mood / character |
|-----|-----------------|
| C major | Pure, simple, innocent, accessible |
| G major | Open, hopeful, pastoral, country |
| D major | Triumphant, bright, anthem |
| A major | Joyful, energetic, confident |
| E major | Brilliant, exciting, royal |
| B major | Powerful, virtuosic |
| F major | Pastoral, calm, comforting |
| Bb major | Warm, romantic, lush |
| Eb major | Heroic, noble, "love" key |
| Ab major | Dreamy, contemplative |
| Db major | Rich, warm, intimate |

## Minor keys (generally darker)

| Key | Mood / character |
|-----|-----------------|
| A minor | Melancholic, contemplative, neutral sad |
| E minor | Sad, longing, yearning, feminine sad |
| D minor | Serious, tragic, "saddest of all keys" (Spinal Tap joke aside) |
| G minor | Brooding, dramatic, dignified sad |
| C minor | Tragic, dark, deeply sad, classical sorrow |
| F minor | Mournful, weeping, deeply sad |
| Bb minor | Funereal, dark beyond sad |
| Eb minor | Cruel, anxious, deeply troubled |
| F# minor | Restless, agitated |
| C# minor | Mournful, romantic darkness |

## Picking key by mood

### Want: bright, happy, anthem
→ D major, A major, E major

### Want: pastoral, calm, comforting
→ F major, G major

### Want: warm, romantic
→ Bb major, Eb major, Ab major

### Want: melancholic but not too dark
→ A minor, E minor

### Want: tragic, sad
→ D minor, G minor, C minor

### Want: deeply sad / mournful
→ F minor, Bb minor

### Want: dark / brooding
→ G minor, C minor, F# minor

## Pick key by vocal range

### Female alto (E3-F5)
Best keys: A, B, C, D, E, F, G major or minor

### Female mezzo (A3-A5)
Best keys: C, D, E, F, G, A major or minor

### Female soprano (C4-C6)
Best keys: D, E, F, G, A, Bb major or minor (higher)

### Male tenor (C3-C5)
Best keys: A, B, C, D, E major or minor

### Male baritone (G2-G4)
Best keys: F, G, A, Bb major or minor

### Male bass (E2-E4)
Best keys: D, E, F, G major or minor (lower)

## Key changes (modulation)

### Up a step (most common)
Final chorus modulates up a whole or half step. Adds energy.
Example: C → D, or C → C#

### Up a 4th
Common in classic pop.
Example: C → F

### Parallel minor/major
Switches between same letter major/minor for mood shift.
Example: C major verse → C minor bridge

## Cultural / regional considerations

### Azerbaijani mugham modes
Mugham uses different scales (modes):
- Rast — major-like, joyful
- Şur — pentatonic-like, melancholic
- Çahargah — mixed, dramatic
- Bayatı-Şiraz — sad, longing
- Segah — tragic, deeply emotional

For Azerbaijani-style music, mention the mugham mode if applicable:
```
[Style: mugham, Şur mode, melancholic]
```

### Other modal music
- Phrygian mode — Spanish, Middle Eastern
- Dorian mode — Celtic, jazz
- Mixolydian — folk, blues

## For AI music (Suno)

Specify key in prompt:
```
[Style: indie folk, 75 BPM, key F minor, melancholic]
```

Suno respects key specifications.

## Common key mistakes

- Key outside vocal range
- Wrong mood for content
- Random key with no purpose
- No key specified (Suno picks randomly)
- Modulation that doesn't fit context
