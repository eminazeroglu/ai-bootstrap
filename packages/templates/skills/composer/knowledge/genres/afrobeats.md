# Afrobeats — Genre

> West African pop genre. Globally popular. Rhythmic, danceable, melodic.

## Defining elements

- West African rhythmic foundation
- Mid-fast tempo (95-120 BPM)
- Pidgin English + local languages (Yoruba, etc.) + English
- Danceable groove
- Live instrumentation hybrid with electronic
- Catchy hooks

## Sub-types / Related

- **Afrobeats (modern)** — Wizkid, Burna Boy, Davido
- **Afrobeat (classic — Fela Kuti)** — different, jazz-funk-political
- **Amapiano** — South African house-influenced (Tyla, modern)
- **Afro-fusion** — mixing with R&B, pop
- **Highlife** — Ghanaian roots
- **Afroswing** — UK African diaspora

## Distinctive elements

- Talking drum or its electronic equivalent
- Polyrhythmic percussion
- Melodic bass lines
- Call-and-response vocals
- Specific dance beats

## Typical instrumentation

Drum machine + percussion (congas, shakers) + bass (melodic) + electric guitar (sometimes funk-style) + synth pads + vocal + horns (sometimes)

## Production characteristics

- Layered percussion
- Bass groove forward
- Vocal forward
- Modern polished production
- Hybrid acoustic-electronic

## Lyrics direction

- Love (modern)
- Celebration
- Dancing
- Cultural pride
- Multi-language code-switching
- Specific local cultural references

```
[Style: modern Afrobeats, 105 BPM, A major, drum machine + layered 
percussion (congas, shakers) + melodic bass + electric guitar + synth 
pads + male vocal with Pidgin English and Yoruba mix, polished hybrid

```

## Examples

- Wizkid, Burna Boy, Davido (Nigerian icons)
- Modern: Tems, Asake, Rema, Tyla (Amapiano)
- Diaspora: Skepta (UK), Drake collaborations
- Classic: Fela Kuti (foundational)

## Avoid

- Single vocal (often features)
- No percussion (groove essential)
- Slow ballad pacing
- Generic "African" (too vague — be specific)
