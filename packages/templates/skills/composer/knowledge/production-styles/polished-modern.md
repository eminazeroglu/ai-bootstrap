# Polished Modern — Production Style

> Clean, professional, radio-ready production. Default for mainstream pop.

## Defining elements

- High audio fidelity
- Clean mix
- Loud mastering (modern loudness war)
- Vocal forward
- Each element clearly placed
- Tight quantization
- Modern processing

## Sound character
- Sleek
- Hi-fi
- Radio-ready
- Sometimes "lifeless" (over-polished)
- Loud throughout

## Genre fit
- Modern pop
- Modern R&B
- Modern hip-hop
- EDM
- Most commercial radio music

## Production techniques
- Heavy compression on most elements
- EQ separation between elements
- Auto-tune (subtle to obvious)
- Sidechain compression (pumping)
- Stereo widening
- Multi-band processing
- Loud mastering (-8 LUFS or louder)

## When to use
- Mainstream commercial production
- Radio targeting
- Streaming optimization (loud = louder than competition)
- Pop / hip-hop / R&B

## For Suno
```
[Production: polished modern pop, professional radio-ready mix, vocal 
forward with subtle auto-tune, tight quantized rhythm, sidechain 
compression, loud mastering]
```

## Avoid pairing with
- Folk / indie acoustic (defeats authentic feel)
- Vintage / nostalgic projects
- Lo-fi aesthetic (opposite)
- Live recording feel
- Classical / jazz acoustic
