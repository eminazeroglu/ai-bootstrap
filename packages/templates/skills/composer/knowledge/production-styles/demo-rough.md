# Demo / Rough — Production Style

> Intentionally unfinished aesthetic. Rough sketch quality. Authentic / vulnerable.

## Defining elements

- Single instrument + voice often
- Phone/laptop recording quality
- No mixing polish
- Imperfections kept
- Vulnerable presentation
- Bedroom / home recording feel

## Sound character
- Raw
- Vulnerable
- Authentic
- Imperfect
- Intimate

## Genre fit
- Bedroom pop / indie
- Acoustic singer-songwriter
- Sketch / demo aesthetic intentionally
- Modern indie (some)
- Voice memo / iPhone recording aesthetic

## Production techniques
- Phone or laptop microphone (intentionally lo-fi)
- No EQ adjustment
- No compression
- Raw vocal performance
- Single take usually
- No vocal tuning
- Background noise audible

## When to use
- Demo / sketch aesthetic
- "Voice memo" production aesthetic
- Vulnerable acoustic moments
- Authentic indie aesthetic
- Behind-the-scenes feel

## Famous demo-style releases
- Phoebe Bridgers acoustic versions
- Many indie demos released as B-sides
- Acoustic versions of pop hits
- "Bedroom recording" aesthetic
- Frank Ocean "channel ORANGE" some moments
- Bon Iver "For Emma, Forever Ago" (cabin recording)

## Modern trend
Many artists release "demo" or "voice memo" versions to feel more intimate / authentic. Modern aesthetic values vulnerability over polish.

## Risks
Can be perceived as low quality if not framed properly. Best as intentional aesthetic choice, not laziness.

## Avoid pairing with
- Mainstream commercial production
- Electronic / EDM
- Polished pop
- Stadium rock
- High-production genres
