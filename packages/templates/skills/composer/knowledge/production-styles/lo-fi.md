# Lo-Fi — Production Style

> Deliberately less polished. Vintage feel, warm imperfection, vinyl crackle.

## Defining elements

- Tape saturation
- Vinyl crackle (intentional)
- Less compression
- Warmer EQ (less brilliance)
- "Imperfect" feel
- Bedroom production aesthetic
- Mid-fi quality (intentionally)

## Sound character
- Warm, fuzzy
- Vintage
- Cozy / intimate
- Imperfect (intentionally)
- Less loud

## Genre fit
- Lo-fi hip-hop / chillhop
- Bedroom pop
- Indie folk (some)
- Vaporwave
- Modern indie

## Production techniques
- Tape emulation plugins (saturation, hiss)
- Vinyl crackle samples / plugins
- Bit reduction (subtle)
- Less compression
- Mid/side EQ for warmth
- Dust / noise textures
- Lower loudness mastering

## When to use
- Lo-fi hip-hop production
- Bedroom indie pop
- Nostalgic production aesthetic
- Chill / study music
- Modern indie aesthetic

## For Suno
```
[Production: lo-fi aesthetic, tape saturation, vinyl crackle, less 
compression, warm vintage feel, bedroom production atmosphere]
```

## Avoid pairing with
- Mainstream pop (loses commercial polish)
- EDM (needs cleanliness for impact)
- Classical (acoustic clarity needed)
- High-energy aggressive music
