# Song Structure — Music Knowledge

> Universal song structure principles. Loaded with every music generation request.

## Standard structures

### Pop / radio (most common)
```
Intro (4-8 bars)
Verse 1 (16 bars)
Pre-Chorus (8 bars, optional)
Chorus (16 bars)
Verse 2 (16 bars)
Pre-Chorus (8 bars)
Chorus (16 bars)
Bridge (8-16 bars)
Final Chorus (16 bars, often louder/elaborated)
Outro (4-8 bars)
```

Total: ~3-3:30 min at 100-120 BPM.

### Hip-hop / rap
```
Intro (4-8 bars)
Verse 1 (16 bars)
Hook/Chorus (8 bars)
Verse 2 (16 bars)
Hook (8 bars)
Verse 3 or Bridge (8-16 bars)
Hook (8 bars)
Outro (4-8 bars)
```

### Electronic / EDM
```
Intro (8-16 bars)
Build-up (8 bars)
Drop (16-32 bars)
Breakdown (8-16 bars)
Build-up
Drop
Outro
```

### Folk / acoustic
```
Verse 1
Chorus
Verse 2
Chorus
Bridge
Final Chorus (often a cappella or stripped)
```

## Section roles

- **Intro** — sets tone, hooks listener
- **Verse** — story / narrative content
- **Pre-Chorus** — builds tension, leads to chorus
- **Chorus** — emotional core, hook, repeated
- **Bridge** — contrast, new perspective, peak
- **Outro** — resolution, fadeout

## Bar counting

Standard pop: 16 bars per section common
4/4 time signature: 4 beats per bar
At 120 BPM: 16 bars = 32 seconds

## Common mistakes

- No clear chorus (audience can't sing along)
- Bridge missing or unmemorable
- Repetition without variation
- Inconsistent section lengths
- Intro too long
- Outro abrupt
