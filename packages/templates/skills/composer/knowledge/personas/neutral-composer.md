# Neutral Composer — Default Persona

> No specific artist signature. Genre-appropriate baseline production. Use when no persona-driven aesthetic wanted.

## When to invoke
- User wants generic genre output (no specific artist style)
- No reference artist matches the brief
- Persona uncertainty — start neutral, refine later
- Educational / template / placeholder use
- Pure genre adherence wanted (not artist twist on genre)

## Approach
- Follow GENRE conventions strictly (use genre file)
- Follow MOOD direction (use mood file)
- Average production quality (not signature lo-fi or hi-fi)
- Standard song structure for genre
- No specific phrasing/vocal mannerisms
- No quirky instrument choices
- "Textbook" version of the genre

## Production defaults
- Polished modern production (default)
- Standard mixing for streaming (LUFS -14)
- Standard arrangement for genre
- Generic vocal style (matching genre conventions)
- Standard BPM range for genre
- Standard key choices for mood

## Vocal defaults
- Match genre convention (no signature artist quirk)
- Clean delivery
- Standard harmonization
- Conventional emotional range

## Lyrical defaults
- Match genre theme conventions
- Universal language (not regional dialect)
- Standard rhyme schemes for genre
- No specific lyricist signature

## For Suno
```
[Style: standard {genre} production with conventional arrangement, 
{mood} atmosphere, {vocal-style} vocal, BPM {standard-for-genre}, 
key {standard-for-mood}, polished modern mix]
```

## When NOT to use
- User specifically references an artist
- User wants distinct stylistic signature
- Brief is highly artistic / experimental
- Album-level conceptual work

## Use case examples
- "Make me a sad pop song" → neutral pop persona + melancholy mood
- "Generate a hip-hop beat" → neutral hip-hop, no Kanye/Pharrell signature
- "Folk song about love" → neutral folk, not Bon Iver / Sufjan flavored

## Pair well with
- Any genre (it's the genre's neutral version)
- Any standard production style
- Any conventional mood
- Mainstream commercial appeal

## Notes
This persona exists to AVOID forcing a famous artist's flavor when the brief doesn't call for it. Sometimes generic IS the right answer.
